Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

040296,000026:                                                                                                  #  PROGRAM NAME:   FINDCDUW
040297,000027: 
040298,000028:                                                                                                  #  MOD NUMBER:     1         68 07 15
040299,000029: 
040300,000030:                                                                                                  #  MOD AUTHOR:     KLUMPP
040301,000031: 
040302,000032:                                                                                                  #  OBJECTS OF MOD: 1.        TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
040303,000033:                                                                                                  #                  2.        TO MAINTAIN CORRECT AND CURRENT THRUST
040304,000034:                                                                                                  #                            DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
040305,000035:                                                                                                  #                            FETCHING FOR THE THRUST DIRECTION FILTER THE
040306,000036:                                                                                                  #                            CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
040307,000037:                                                                                                  #                            MODES.
040308,000038:                                                                                                  #                  3.        TO SUBSTITUTE A STOPRATE FOR THE NORMAL
040309,000039:                                                                                                  #                            AUTOPILOT COMMANDS WHENEVER
040310,000040:                                                                                                  #                            1) NOT IN PNGCS-AUTO, OR
040311,000041:                                                                                                  #                            2) ENGINE IS OFF.
040312,000042: 
040313,000043:                                                                                                  #  FUNCTIONAL DESCRIPTION:
040314,000044: 
040315,000045:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
040316,000046:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
040317,000047:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
040318,000048:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
040319,000049:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
040320,000050:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
040321,000051:                                                                                                  #  ACCELERATIONS AVAILABLE).
040322,000052: 
040323,000053:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
040324,000054:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN  XOVINHIB  SET,
040325,000055:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

040328,000058:                                                                                                  #  SPECIFICATIONS:
040329,000059: 
040330,000060:                                                                                                  #  INITIALIZATION: A SINGLE INTERPRETIVE CALL TO  INITCDUW  IS REQUIRED
040331,000061:                                                                                                  #                  BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
040332,000062: 
040333,000063:                                                                                                  #  CALL:           INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
040334,000064:                                                                                                  #                  VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
040335,000065:                                                                                                  #                  THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
040336,000066: 
040337,000067:                                                                                                  #  RETURNS:        NORMAL INTERPRETIVE IN ALL CASES
040338,000068: 
040339,000069:                                                                                                  #                  1.       NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
040340,000070: 
040341,000071:                                                                                                  #                  2.       IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
040342,000072:                                                                                                  #                           WITHOUT ISSUING AUTOPILOT CMDS.
040343,000073: 
040344,000074:                                                                                                  #                  3.       IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
040345,000075:                                                                                                  #                           ISSUING AUTOPILOT CMDS.
040346,000076: 
040347,000077:                                                                                                  #  ALARMS:         00401 IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
040348,000078:                                                                                                  #                           FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
040349,000079:                                                                                                  #                           BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
040350,000080:                                                                                                  #                  00402 IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
040351,000081:                                                                                                  #                           UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
040352,000082:                                                                                                  #                           STOPRATE AS ONLY INPUT TO AUTOPILOT.
040353,000083: 
040354,000084:                                                                                                  #  INPUTS:         UNFC/2   THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
040355,000085:                                                                                                  #                  UNWC/2   WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
040356,000086:                                                                                                  #                  XOVINHIB FLAG DENOTING X AXIS OVERRIDE INHIBITED.
040357,000087:                                                                                                  #                  CSMDOCKD FLAG DENOTING CSM DOCKED.
040358,000088:                                                                                                  #                  STEERSW  FLAG DENOTING INSUFF THRUST FOR THRUST DIR FLTR.
040359,000089: 
040360,000090:                                                                                                  #  OUTPUTS:        DELCDUX,Y,Z
040361,000091:                                                                                                  #                  OMEGAPD,+1,+2
040362,000092:                                                                                                  #                  DELPEROR,+1,+2
040363,000093:                                                                                                  #                  CPHI,+1,+2 FOR NOUN22
040364,000094: 
040365,000095:                                                                                                  #  DEBRIS:         FINDCDUW DESTROYS SINCDUX,Y,Z AND COSCDUX,Y,Z BY
040366,000096:                                                                                                  #                  WRITING INTO THESE LOCATIONS THE SINES AND COSINES
040367,000097:                                                                                                  #                  OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
