Source Code

These source-code files are part of a reconstructed copy of LM131 revision 1, the final release of the Apollo 13 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The source reconstruction began with source code of Luminary 131 previously transcribed from a digitized copy of that program. A dump of the flight spare core rope memory module B5, part number 2010802-171, serial number RAY 411, was obtained. This module is the only module different between Luminary 131 and LM131 revision 1. The dump was disassembled and all changes were incorporated into these source files. Note that page numbers in the reconstructed code match those on the Luminary 131 printout, although the changed code would likely have changed page numbers for a real LM131 revision 1 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

038776,000022:                                                                                                  #  PROGRAM NAME:  FINDCDUW
038777,000023:                                                                                                  #  MOD NUMBER:  1        68 07 15
038778,000024:                                                                                                  #  MOD AUTHOR:  KLUMPP
038779,000025: 
038780,000026:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
038781,000027:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
038782,000028:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
038783,000029:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
038784,000030:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
038785,000031:                                                                                                  #                                MODES.
038786,000032:                                                                                                  #                        3.      TO SUBSTITUTE A STOPRATE FOR THE NORMAL
038787,000033:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
038788,000034:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
038789,000035:                                                                                                  #                                2) ENGINE IS OFF.
038790,000036: 
038791,000037:                                                                                                  #  FUNCTIONAL DESCRIPTION:
038792,000038: 
038793,000039:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
038794,000040:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
038795,000041:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
038796,000042:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
038797,000043:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
038798,000044:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
038799,000045:                                                                                                  #  ACCELERATIONS AVAILABLE).
038800,000046: 
038801,000047:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
038802,000048:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XOVINHIB SET,
038803,000049:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

038820,000066:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
