Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

039797,000016:                                                                                                  #  NAME....TVCRESTART PACKAGE,  CONSISTING OF REDOTVC, ENABL1, 2, CMDSOUT, PHSCHK2, ETC.
039798,000017:                                                                                                  #  LOG SECTION....TVCRESTART PACKAGE    SUBROUTINE....DAPCSM
039799,000018:                                                                                                  #  MOD BY ENGEL                         DATE....19 OCT, 1967
039800,000019: 
039801,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION....
039802,000021: 
039803,000022:                                                                                                  #      *RESTART-PROOFS THE TVC DAPS, INCLUDING PITCHDAP, YAWDAP,
039804,000023:                                                                                                  #       TVCEXECUTIVE, ROLL DAP, TVCINIT4, TVCDAPON, AND STROKE TEST
039805,000024:                                                                                                  #      *TVC RESTARTS DESERVE SPECIAL CONSIDERATION IN SEVERAL AREAS.
039806,000025:                                                                                                  #       RESTART DOWN-TIME IS IMPORTANT BECAUSE OF THE TRANSIENTS INTRODUCED
039807,000026:                                                                                                  #       BY THE THRUST VECTOR RETURN TO THE ACTUATOR MECHANICAL NULLS
039808,000027:                                                                                                  #       FOLLOWING TVC- AND OPTICS-ERROR-COUNTER-DISENABLES (CHANNEL 12).
039809,000028:                                                                                                  #       TVC    USES A MIXTURE OF WAITLIST, T5, T6, AND JOB CALLS. THERE IS
039810,000029:                                                                                                  #       FILTER MEMORY (UP TO 7TH ORDER) TO BE PROTECTED IF WILD TRANSIENTS
039811,000030:                                                                                                  #       ARE TO BE AVOIDED. SEVERAL COUNTERS ARE INVOLVED FOR TIMING TVC
039812,000031:                                                                                                  #       EVENTS SUCH AS SWITCHOVER AND STROKE TEST STARTUPS AND RE-STARTUPS.
039813,000032:                                                                                                  #       THE TVC GAINS ARE DECREMENTED. THE GIMBAL TRIM ESTIMATORS AND THE
039814,000033:                                                                                                  #       BODY AXIS ATTITUDE ERROR INTEGRATORS INVOLVE DIGITAL SUMMATION.
039815,000034:                                                                                                  #       DIGITAL DIFFERENTIATORS ARE INVOLVED IN THE BODY AXIS RATE ESTIMA-
039816,000035:                                                                                                  #       TIONS AND IN THE OUTPUTTING OF ACTUATOR COMMANDS. THERE IS AN
039817,000036:                                                                                                  #       OFFSET-TRACKER-FILTER TO PROTECT. ETC., ETC.
039818,000037:                                                                                                  #      *THOSE QUANTITIES WHICH MUST BE PROTECTED ARE STORED IN TEMPORARY
039819,000038:                                                                                                  #       REGISTERS AS THEY ARE COMPUTED, FOR UPDATING THE REAL REGISTERS
039820,000039:                                                                                                  #       DURING COPYCYCLES.
039821,000040:                                                                                                  #      *THE SEVERAL COPYCYCLES ARE EACH PROTECTED BY PHASE POINTS AT THEIR
039822,000041:                                                                                                  #       BEGINNING AND AT THEIR TERMINATION. THE PHASE POINTS ARE SIMPLY
039823,000042:                                                                                                  #       ..INCR.. INSTRUCTIONS,   EITHER ..INCR TVCEXPHS.. FOR COPYCYCLES
039824,000043:                                                                                                  #       IN THE TVCEXECUTIVE, OR ..INCR TVCPHASE.. FOR THE PITCH AND YAW
039825,000044:                                                                                                  #       COPYCYCLES. INDEXING ON EACH OF THESE POINTERS THEN PERMITS A
039826,000045:                                                                                                  #       RETURN TO THE APPROPRIATE RESTART POINTS.
039827,000046:                                                                                                  #      *IF A RESTART OCCURS DURING EITHER COPYCYCLE, THAT COPYCYCLE IS
039828,000047:                                                                                                  #       COMPLETED. THEN THE NORMAL TVCINIT4....DAPINIT....PITCHDAP STARTUP
039829,000048:                                                                                                  #       SEQUENCE IS CALLED UPON TO GET THINGS GOING AGAIN.
039830,000049:                                                                                                  #      *TVC-ENABLE AND OPTICS-ERROR-COUNTER ENABLE MUST BE SET ASAP
039831,000050:                                                                                                  #       (ALLOWING FOR PROCEDURAL DELAYS). THEN THE ENGINES ARE COMMANDED
039832,000051:                                                                                                  #       TO THE P,YACTOFF TRIM VALUES. THE DAPS ARE THEN READY TO GO ON THE
039833,000052:                                                                                                  #       AIR, WITH THE REGULAR STARTUP SEQUENCE, EITHER AT MRCLEAN FOR A
039834,000053:                                                                                                  #       COMPLETE INITIALIZATION OR AT TVCINIT4 FOR A PARTIAL INITIALIZATION
039835,000054:                                                                                                  #      *FOR RESTARTS PRIOR TO THE SETTING OF THE T5 BITS IN IGNOVER THE
039836,000055:                                                                                                  #       PRE40.6 SECTION OF S40.6 TAKES CARE OF RE-ESTABLISHING TRIMS.
039837,000056:                                                                                                  #      *IF A RESTART OCCURS DURING THE TVCEXEC....TVCEXFIN SEQUENCE THE
039838,000057:                                                                                                  #       COMPUTATIONS WILL BE COMPLETED, STARTING AT THE APPROPRIATE RESTART
039839,000058:                                                                                                  #       POINT, AFTER THE DAPS ARE READY TO GO ON THE AIR.
039840,000059:                                                                                                  #      *IF A RESTART OCCURS PRIOR TO TVCINIT4 (TVCPHASE = -1) E.G. DURING
039841,000060:                                                                                                  #       THE EARLY DAP INITIALIZATION PHASE, THE DAP STARTUP SEQUENCE IS
039842,000061:                                                                                                  #       ENTERED AT MRCLEAN FOR A FULL INITIALIZATION.
039843,000062:                                                                                                  #      *RESTARTS ARE NOT CRITICAL TO THE ROLL DAP PERFORMANCES HENCE THE
039844,000063:                                                                                                  #       THE ROLL DAP IS MERELY RESTARTED.
039845,000064:                                                                                                  #      *RESTARTS DURING A STROKE TEST (STROKER IS NON-ZERO) WILL CAUSE THE
039846,000065:                                                                                                  #       STROKE TEST TO BE TERMINATED. A NEW V68 ENTRY WILL BE REQUIRED

039848,000067:                                                                                                  #       TO GET IT GOING AGAIN (NO AUTOMATIC RESTART).
039849,000068:                                                                                                  #      *REDOTVC IS REACHED FOLLOWING ANY RESTART WHICH FINDS THE T5 BITS
039850,000069:                                                                                                  #       (BITS 15,14 OF FLAGWRD6) SET FOR TVC. IGNOVER PREPARES TVCPHASE =-1

039853,000072:                                                                                                  #       TO CALL RCSUP/RCSDAPON WHICH RESETS THE T5 BITS(FOR RCS) ON A
039854,000073:                                                                                                  #       NORMAL SHUTDOWN.
