Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050522,000032:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050523,000033:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050524,000034:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2. THE MEANING OF THESE LISTS
050525,000035:                                                                                                  #     FOLLOW.
050526,000036: 
050527,000037:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050528,000038: 
050529,000039: 
050530,000040:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050531,000041:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050532,000042:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050533,000043:                                                                                                  #                                          .
050534,000044:                                                                                                  #                                          .
050535,000045:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050536,000046:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050537,000047: 
050538,000048:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050539,000049:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050540,000050:                                                                                                  #                                          .
050541,000051:                                                                                                  #                                          .
050542,000052:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050543,000053:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050544,000054: 
050545,000055:                                                                                                  #  WARNINGS-
050546,000056:                                                                                                  #  --------
050547,000057:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
050548,000058:                                                                                                  #               2) 9 TASKS MAXIMUM
050549,000059:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050550,000060:                                                                                                  #               4) TASKS END BY TC TASKOVER
050551,000061: 
050552,000062:                                                                                                  #  CALLING SEQUENCE-
050553,000063: 
050554,000064:                                                                                                  #               L-1    CA     DELTAT   (TIME IN CENTISECONDS TO TASK START)
050555,000065:                                                                                                  #               L      TC     WAITLIST
050556,000066:                                                                                                  #               L+1    2CADR  DESIRED TASK
050557,000067:                                                                                                  #               L+2    (MINOR OF 2CADR)
050558,000068:                                                                                                  #               L+3    RELINT          (RETURNS HERE)

050563,000073:                                                                                                  #               SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

050590,000100: 
050591,000101:                                                                                                  #               CENTRALS- A,Q,L
050592,000102:                                                                                                  #               OTHER   - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050593,000103:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050594,000104:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050595,000105:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
050596,000106: 
050597,000107:                                                                                                  #               LET TO = THE TIME OF THE TC WAITLIST
050598,000108:                                                                                                  #               LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
050599,000109:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050600,000110:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050601,000111:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050602,000112:                                                                                                  #     (Z=0, USUALLY)
050603,000113:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050604,000114:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050605,000115:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050606,000116: 
050607,000117: 
050608,000118:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050609,000119:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050610,000120:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
