Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

040921,000016:                                                                                                  #  NAME     STROKE TEST PACKAGE   (INCLUDING INITIALIZATION PACKAGE)
040922,000017:                                                                                                  #  ORIGINAL CODING BY OLSSON       LOG SECTION....STROKE TEST PACKAGE
040923,000018:                                                                                                  #  MOD BY ENGEL                    DATE....21 MARCH, 1967
040924,000019: 
040925,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION....
040926,000021:                                                                                                  #       STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
040927,000022:                                                                                                  #       STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
040928,000023:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
040929,000024:                                                                                                  #                THEN TESTS FOR AN 80MS DAP. IF 80MS IT SETS STROKER = ESTROKER
040930,000025:                                                                                                  #                FOR AN IMMEDIATE STROKE TEST, OTHERWISE IT MERELY ENABLES
040931,000026:                                                                                                  #                A STROKE TEST BY SETTING STROKER TO -0. THE STROKE TEST
040932,000027:                                                                                                  #                THEN AWAITS SWITCHOVER TO THE 80MS DAP WHEREUPON IT IS
040933,000028:                                                                                                  #                ENABLED AFTER AN ADDITIONAL 4 SECOND DELAY TO AVOID
040934,000029:                                                                                                  #                THE SWITCHOVER TRANSIENTS (SEE STRKCALL, STRKUP IN
040935,000030:                                                                                                  #                TVCEXECUTIVE)
040936,000031:                                                                                                  #       HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
040937,000032:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
040938,000033:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
040939,000034:                                                                                                  #                TVC DAPS, WITH INTERMEDIATE WAITLIST CALLS. NOTE, HOWEVER
040940,000035:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
040941,000036:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
040942,000037:                                                                                                  #                TEST PARAMETER SELECTION
040943,000038:                                                                                                  #       RESTARTS CAUSE TEST TO BE TERMINATED. ANOTHER V68 REQD IF TEST
040944,000039:                                                                                                  #                IS TO BE RE-RUN.
040945,000040:                                                                                                  #       PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
040946,000041:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
040947,000042:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
040948,000043:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
040949,000044:                                                                                                  #                NOMINALLY 50/42.15  =  1.185 DEG
040950,000045:                                                                                                  #  CALLING SEQUENCE....
040951,000046:                                                                                                  #       EXTENDED VERB 68 SETS UP STRKTSTI JOB
040952,000047:                                                                                                  #       PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A  ..TC HACK..
040953,000048:                                                                                                  #       AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT  ..HACKWLST..
040954,000049:                                                                                                  #  NORMAL EXIT MODES....
040955,000050:                                                                                                  #       TC BUNKER  (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST)LIST
040956,000051:                                                                                                  #  SUBROUTINES CALLED....
040957,000052:                                                                                                  #       WAITLIST
040958,000053:                                                                                                  #  ALARM OR ABORT EXIT MODES....
040959,000054:                                                                                                  #       NONE
040960,000055:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
040961,000056:                                                                                                  #       ESTROKER (PAD-LOAD)
040962,000057:                                                                                                  #       STROKER,  CADDY,  REVS,  CARD,  N
040963,000058:                                                                                                  #  OUTPUT....
040964,000059:                                                                                                  #       STRKTSTI...INITIALIZATION FOR STROKE TEST
040965,000060:                                                                                                  #       HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA..ADS..
040966,000061:                                                                                                  #                        RESETS STROKER = +0 WHEN TEST COMPLETED
040967,000062:                                                                                                  #  DEBRIS....
040968,000063:                                                                                                  #        N = CADDY = +0, CARD = -0, REVS = -1
040969,000064:                                                                                                  #        BUNKER
