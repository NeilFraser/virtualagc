Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

042524,000261:                                                                                                  #           CDUXD = CDUXD + DELCDUX    (DOUBLE PRECISION)
042525,000262:                                                                                                  #           CDUYD = CDUYD + DELCDUY    (DOUBLE PRECISION)
042526,000263:                                                                                                  #           CDUZD = CDUZD + DELCDUZ    (DOUBLE PRECISION)
042527,000264: 
042528,000265:                                                                                                  #  THE STEERING PROGRAMS-
042529,000266:                                                                                                  #           1) ATTITUDE MANEUVER ROUTINE
042530,000267:                                                                                                  #           2) LEM TRACKING
042531,000268: 
042532,000269:                                                                                                  #  SHOULD GENERATE THE DESIRED ANGLES (CDUXD, CDUYD, CDUZD) AS WELL AS THE INCREMENTAL ANGLES (DELCDUX, DELCDUY,
042533,000270:                                                                                                  #  DELCDUZ) SO THAT THE GIMBAL ANGLE COMMANDS CAN BE INTERPOLATED BETWEEN UPDATES.
042534,000271: 
042535,000272:                                                                                                  #  HOLDFLAG CODING-
042536,000273: 
042537,000274:                                                                                                  #      +  = GRAB PRESENT CDU ANGLES AND STORE IN THETADX, THETADY, THETADZ
042538,000275:                                                                                                  #           AND PERFORM ATTITUDE HOLD ABOUT THESE ANGLES
042539,000276:                                                                                                  #           ALSO IGNORE AUTOMATIC STEERING

042541,000278:                                                                                                  #                  1) INITIALIZATION PHASE OF AUTOPILOT
042542,000279:                                                                                                  #                  2) OCCURANCE OF RHC COMMANDS

042546,000283:                                                                                                  #                     HOLDFLAG IS NOT ACTUALLY SET TO +, BUT THE LOGIC
