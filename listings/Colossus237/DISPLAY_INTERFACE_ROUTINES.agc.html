Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

060090,000022:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES-
060091,000023: 
060092,000024:                                                                                                  #      1. PRIORITY DISPLAYS- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS. USUALLY THESE DISPLAYS ARE SENT
060093,000025:                                                                                                  #         OUT UNDER CRITICAL ALARM CONDITIONS.
060094,000026:                                                                                                  #      2. EXTENDED VERB DISPLAYS- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
060095,000027:                                                                                                  #      3. NORMAL DISPLAYS- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
060096,000028:                                                                                                  #         SEQUENCE OF EVENTS.
060097,000029:                                                                                                  #      4. MISC. DISPLAYS- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACEROUTINES. THESE INCLUDE SUCH DISPLAYS AS
060098,000030:                                                                                                  #         MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
060099,000031: 
060100,000032: 
060101,000033:                                                                                                  #      5. ASTRONAUT INITIATED DISPLAYS- ALL DISPLAYS INITIATED EXTERNALLY.
060102,000034:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
060103,000035: 
060104,000036:                                                                                                  #      1. ACTIVE-THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
060105,000037:                                                                                                  #         (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
060106,000038:                                                                                                  #         BUT NOT WAITING FOR A RESPONSE.
060107,000039:                                                                                                  #      2. INACTIVE -A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTEDBY A DISPLAY OF HIGHER PRIORITY,
060108,000040:                                                                                                  #         (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
060109,000041:                                                                                                  #         WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
060110,000042:                                                                                                  #         OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
060111,000043:                                                                                                  #         HAS INFO SAVED FOR RESTART PURPOSES.
060112,000044: 
060113,000045: 
060114,000046:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS-
060115,000047: 
060116,000048:                                                                                                  #    INTERRUPTS-
060117,000049:                                                                                                  #      1. THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
060118,000050:                                                                                                  #      2. INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
060119,000051:                                                                                                  #      3. PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS. A PRIORITY DISPLAY INTERRUPTING ANOTHER
060120,000052:                                                                                                  #         PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
060121,000053:                                                                                                  #      4. A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
060122,000054:                                                                                                  #      5. A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
060123,000055: 
060124,000056:                                                                                                  #    ORDER OF WAITING DISPLAYS-
060125,000057:                                                                                                  #      1. ASTRONAUT EXTERNAL USE
060126,000058:                                                                                                  #      2. PRIORITY
060127,000059:                                                                                                  #      3. INTERRUPTED MARK

060135,000067:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
060136,000068:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
060137,000069:                                                                                                  #  WILL FOLLOW LATER):
060138,000070: 
060139,000071:                                                                                                  #       1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
060140,000072:                                                                                                  #           IS TO THE USERS CALL CADR +4.  FOR THE ONLY NON FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
060141,000073:                                                                                                  #           IS TO THE USERS CALLING LOC +1.
060142,000074:                                                                                                  #       2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.
060143,000075:                                                                                                  #       3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USERS JOB PRIORITY.
060144,000076:                                                                                                  #       4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
060145,000077:                                                                                                  #           USERS JOB.
060146,000078:                                                                                                  #       5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
060147,000079:                                                                                                  #       6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
060148,000080:                                                                                                  #           DESCRIPTION OF RESTARTS).
060149,000081:                                                                                                  #       7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPONSES.  A TERMINATE (V34) BRANCHES
060150,000082:                                                                                                  #           TO THE USERS CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USERS CALL CADR +2.  AN ENTER OR RECYCLE
060151,000083:                                                                                                  #           (V32) BRANCHES TO THE USERS CALL CADR +3.
060152,000084:                                                                                                  #       8.  ALL ROUTINES MUST BE USED UNDER EXECUTIVE CONTROL.
