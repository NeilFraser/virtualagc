Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

045973,000022:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
045974,000023:                                                                                                  #  MOD NO.- 0      TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
045975,000024:                                                                                                  #                  PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
045976,000025:                                                                                                  #                  THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.REFER TO DOWNLINK
045977,000026:                                                                                                  #                  LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
045978,000027:                                                                                                  #                  OF DOWN TELEMETRY DATA.
045979,000028:                                                                                                  #  MOD BY-  KILROY, SMITH, DEWITT
045980,000029:                                                                                                  #  DATE-    02OCT67
045981,000030:                                                                                                  #  AUTHORS- KILROY, SMITH, DEWITT, DEWOLF, FAGIN
045982,000031:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
045983,000032:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
045984,000033:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
045985,000034:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
045986,000035:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
045987,000036:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
045988,000037:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
045989,000038:                                                                                                  #           LGC IN SERIAL FASHION.
045990,000039:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
045991,000040:                                                                                                  #           OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
045992,000041:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
045993,000042:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
045994,000043:                                                                                                  #  CALLING SEQUENCE- NONE
045995,000044:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
045996,000045:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
045997,000046:                                                                                                  #           IN EFFECT IS A RESUME.
045998,000047:                                                                                                  #  SUBROUTINES CALLED- NONE
045999,000048:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
046000,000049:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
046001,000050:                                                                                                  #  RESTART PROTECTION:
046002,000051:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER(ACTUALLY
046003,000052:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST(I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
046004,000053:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
046005,000054:                                                                                                  #           THE RESTART(OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
046006,000055:                                                                                                  #           DOWNLIST.
046007,000056:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
046008,000057:                                                                                                  #  DOWNLINK LIST SELECTION:
046009,000058:                                                                                                  #           THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
046010,000059:                                                                                                  #           1. FRESH START
046011,000060:                                                                                                  #           2. V37EXXE  WHERE XX = THE MAJOR MODE BEING SELECTED. 
046012,000061:                                                                                                  #           3. UPDATE PROGRAM(P27)
046013,000062:                                                                                                  #           4.     NON-V37 SELECTABLE TYPE PROGRAMS(E.G. AGS INITIALIZATION(SUNDANCE,LUMINARY) AND P61-P62
046014,000063:                                                                                                  #                  TRANSITION(COLOSSUS) ETC.).
046015,000064:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
046016,000065:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
046017,000066:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E. 200 LGC
046018,000067:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
046019,000068:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
046020,000069:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
046021,000070:                                                                                                  #  DEBRIS(ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
046022,000071:                                                                                                  #           LDATALST,DNTMBUFF TO DNTMBUFF +21D,TMINDEX,DNQ.

046027,000076:                                                                                                  #  DODOWNTM IS ENTERED EVERY 20 MS BY AN INTERRUPT TRIGGERED BY THE
046028,000077:                                                                                                  #  RECEIPT OF AN ENDPULSE FROM THE SPACECRAFT TELEMETRY PROGRAMMER.
046029,000078: 
046030,000079:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
046031,000080:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
046032,000081:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
046033,000082:                                                                                                  #     EXECUTION.
046034,000083:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
046035,000084:                                                                                                  #     SUBLISTS.
046036,000085:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
046037,000086:                                                                                                  #     DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
046038,000087:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
046039,000088:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
046040,000089:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
046041,000090:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
046042,000091:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
046043,000092:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
046044,000093:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
046045,000094:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
046046,000095:                                                                                                  #     A. 1DNADR              1-WORD DOWNLIST ADDRESS.
046047,000096:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
046048,000097:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
046049,000098:                                                                                                  #     B. 2DNADR - 6DNADR     N-WORD DOWNLIST ADDRESS, N = 2 - 6.
046050,000099:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
046051,000100:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
046052,000101:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
046053,000102:                                                                                                  #     C. DNCHAN              DOWNLIST CHANNEL ADDRESS.
046054,000103:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
046055,000104:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
046056,000105:                                                                                                  #     D. DNPTR               DOWN TELEMETRY SUBLIST POINTER.
046057,000106:                                                                                                  #      SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
046058,000107:                                                                                                  #      CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
046059,000108:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
046060,000109:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST(ONLY).
046061,000110:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

046071,000120:                                                                                                  #     EQUIVALENT OF THE FOLLOWING ECADRS(I.E. IDNADRS): 377, 777, 1377, 1777, 2377, 27777, 3377, 3777.

046276,000325:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND(DUMP) ALL ERASABLE STORAGE 'N' TIMES.(N = 1 TO 4). BANKS ARE SENT ONE AT A TIME

046280,000329:                                                                                                  #           AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
046281,000330:                                                                                                  #           WILL BEGIN.
