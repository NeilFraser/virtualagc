Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

007199,000047:                                                                                                  #  DOWNLIST FORMAT DEFINITIONS AND RULES-
007200,000048:                                                                                                  #  1. END OF A LIST = -XDNADR (X = 1 TO 6), -DNPTR, OR -DNCHAN.
007201,000049:                                                                                                  #  2. SNAPSHOT SUBLIST = LIST WHICH STARTS WITH A -1DNADR.
007202,000050:                                                                                                  #  3. SNAPSHOT SUBLIST CAN ONLY CONTAIN 1DNADRS.
007203,000051:                                                                                                  #  4. TIME2 1DNADR MUST BE LOCATED IN THE CONTROL LIST OF A DOWNLIST.
007204,000052:                                                                                                  #  5. ERASABLE DOWN TELEMETRY WORDS SHOULD BE GROUPED IN SEQUENTIAL
007205,000053:                                                                                                  #    LOCATIONS AS MUCH AS POSSIBLE TO SAVE STORAGE USED BY DOWNLINK LISTS.
007206,000054:                                                                                                  #  6. THE DOWNLINK LISTS(INCLUDING SUBLISTS) ARE ORGANIZED SUCH THAT THE ITEMS LISTED FIRST(IN FRONT OF FBANK) ARE
007207,000055:                                                                                                  #     SENT FIRST.  EXCEPTION--- SNAPSHOT SUBLISTS. IN THE SNAPSHOT SUBLISTS THE DATA REPRESENTED BY THE FIRST
007208,000056:                                                                                                  #     11 1DNADRS IS PRESERVED (IN ORDER) IN DNTMBUFF AND SENT BY THE NEXT 11 DOWNRUPTS. THE DATA REPRESENTED BY THE
007209,000057:                                                                                                  #     LIST IS SENT IMMEDIATELY.
