Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Linden Sims. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 56 OF AGC PROGRAM ZERLINA BY ZOROASTER
	9:12 OCT. 21,1970
Note that the date is the date of the printout, not the date of the program revision.

060159,000033:                                                                                                  #  FUNCT. DESCR:   P27 (THE UPDATE PROGRAM) PROCESSES COMMANDS AND DATA

060174,000048:                                                                                                  #  RESTARTS:       P27 IS RESTART PROTECTED IN TWO WAYS...
060175,000049:                                                                                                  #                  1. PRIOR TO VERIFLAG INVERSION(WHICH IS CAUSED BY THE GROUND/ASTRONAUT'S VERIFICATION OF UPDATE
060176,000050:                                                                                                  #                     DATA BY SENDING A V33E WHEN V21N02 IS FLASHING)---
060177,000051:                                                                                                  #                     NO PROTECTION EXCEPT PRE-P27 MODE IS RESTORED, COAST + ALIGN DOWNLIST IS SELECTED AND UPLINK
060178,000052:                                                                                                  #                     ACTIVITY LIGHT IS TURNED OFF.(JUST AS IF A V34E WAS SENT DURING P27 DATA LOADS).
060179,000053:                                                                                                  #                     V70,V71,V72 OR V73 WILL HAVE TO BE COMPLETELY RESENT BY USER.
060180,000054:                                                                                                  #                  2. AFTER VERIFLAG INVERSION(WHEN UPDATE OF THE SPECIFIED ERASABLES IS BEING PERFORMED)---
060181,000055:                                                                                                  #                     PROTECTED AGAINST RESTARTS.
060182,000056: 
060183,000057:                                                                                                  #  DEBRIS:         UPBUFF   (20D)  TEMP STORAGE FOR ADDRESSES AND CONTENTS.
060184,000058:                                                                                                  #                  UPVERB   (1)    VERB NUMBER MINUS 70D (E.G. FOR V72, UPVERB = 72D - 70D = 2)
060185,000059:                                                                                                  #                  UPOLDMOD (1)    FOR MAJOR MODE INTERRUPTED BY P27.
060186,000060:                                                                                                  #                  COMPNUMB (1)    TOTAL NUMBER OF COMPONENTS TO BE TRANSMITTED.
060187,000061:                                                                                                  #                  UPCOUNT  (1)    ACTUAL NUMBER OF COMPONENTS RECEIVED.
060188,000062:                                                                                                  #                  UPTEMP   (1)    SCRATCH, BUT USUALLY CONTAINS COMPONENT NUMBER TO BE CHANGED DURING VERIFY CYCLE
