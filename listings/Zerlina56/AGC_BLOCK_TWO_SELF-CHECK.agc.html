The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. 055688,000029:                                                                                                  #  FUNCTIONAL DESCRIPTION
055689,000030: 
055690,000031:                                                                                                  #       PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
055691,000032:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
055692,000033:                                                                                                  #  STARTING VERB.
055693,000034:                                                                                                  #       THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
055694,000035:                                                                                                  #       THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK , ONE AT A TIME.
055695,000036:                                                                                                  #       IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
055696,000037:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
055697,000038:                                                                                                  #       THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
055698,000039:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
055699,000040:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
055700,000041:                                                                                                  #  +-4   ERASABLE MEMORY
055701,000042:                                                                                                  #  +-5   FIXED MEMORY
055702,000043:                                                                                                  #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
055703,000044:                                                                                                  #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
055704,000045:                                                                                                  #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
055705,000046: 
055706,000047: 
055707,000048:                                                                                                  #  WARNINGS
055708,000049: 
055709,000050:                                                                                                  #       USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
055710,000051:                                                                                                  #       SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
055711,000052: 
055712,000053: 
055713,000054:                                                                                                  #  CALLING SEQUENCE
055714,000055: 
055715,000056:                                                                                                  #       TO CALL SELF-CHECK KEY IN
055716,000057:                                                                                                  #            V 21 N 27 E  OPTION NUMBER E
055717,000058:                                                                                                  #       TO CALL SHOW-BANKSUM KEY IN
055718,000059:                                                                                                  #            V 91 E         DISPLAYS FIRST BANK
055719,000060:                                                                                                  #            V 33 E         PROCEED, DISPLAYS NEXT BANK
055720,000061: 
055721,000062: 
055722,000063:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
055723,000064: 
055724,000065:                                                                                                  #       SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
055725,000066:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
055726,000067:                                                                                                  #       THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
055727,000068:                                                                                                  #       SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
055728,000069: 
055729,000070: 
055730,000071: 
055731,000072:                                                                                                  #  OUTPUT

055733,000074:                                                                                                  #       SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
055734,000075:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
055735,000076:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
055736,000077:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
055737,000078:                                                                                                  #  INITIATED FRESH START (SLAP1).
055738,000079:                                                                                                  #       SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
055739,000080:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
055740,000081: 
055741,000082: 
055742,000083:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
055743,000084: 
055744,000085:                                                                                                  #       ACCOMPLISHED BY FRESH START
055745,000086:                                                                                                  #            SMODE SET TO +0
055746,000087: 
055747,000088: 
055748,000089:                                                                                                  #  DEBRIS
055749,000090: 
055750,000091:                                                                                                  #       ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
055751,000092:                                                                                                  #  EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
055752,000093:                                                                                                  #  E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
