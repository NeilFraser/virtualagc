Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Linden Sims. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

043789,000023:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
043790,000024:                                                                                                  #  MOD NO.- 0      TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
043791,000025:                                                                                                  #                  PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
043792,000026:                                                                                                  #                  THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.REFER TO DOWNLINK
043793,000027:                                                                                                  #                  LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
043794,000028:                                                                                                  #                  OF DOWN TELEMETRY DATA.
043795,000029:                                                                                                  #  MOD BY-  KILROY, SMITH, DEWITT
043796,000030:                                                                                                  #  DATE-    02OCT67
043797,000031:                                                                                                  #  AUTHORS- KILROY, SMITH, DEWITT, DEWOLF, FAGIN
043798,000032:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
043799,000033:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
043800,000034:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
043801,000035:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
043802,000036:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
043803,000037:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
043804,000038:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
043805,000039:                                                                                                  #           LGC IN SERIAL FASHION.
043806,000040:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
043807,000041:                                                                                                  #           OCCUR EVERY 20MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
043808,000042:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
043809,000043:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
043810,000044:                                                                                                  #  CALLING SEQUENCE- NONE
043811,000045:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
043812,000046:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
043813,000047:                                                                                                  #           IN EFFECT IS A RESUME.
043814,000048:                                                                                                  #  SUBROUTINES CALLED- NONE
043815,000049:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
043816,000050:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
043817,000051:                                                                                                  #  RESTART PROTECTION:
043818,000052:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER(ACTUALLY
043819,000053:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST(I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
043820,000054:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
043821,000055:                                                                                                  #           THE RESTART(OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
043822,000056:                                                                                                  #           DOWNLIST.
043823,000057:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
043824,000058:                                                                                                  #  DOWNLINK LIST SELECTION:
043825,000059:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
043826,000060:                                                                                                  #        1. FRESH START
043827,000061:                                                                                                  #        2. V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
043828,000062:                                                                                                  #        3. UPDATE PROGRAM(P27)
043829,000063:                                                                                                  #        4.     NON-V37 SELECTABLE TYPE PROGRAMS(E.G. AGS INITIALIZATION(SUNDANCE,LUMINARY) AND P61-P62
043830,000064:                                                                                                  #               TRANSITION(COLOSSUS) ETC.).
043831,000065:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
043832,000066:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
043833,000067:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E. 200 LGC
043834,000068:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
043835,000069:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
043836,000070:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
043837,000071:                                                                                                  #  DEBRIS(ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
043838,000072:                                                                                                  #           LDATALST,DNTMBUFF TO DNTMBUFF +21D,TMINDEX,DNQ.

043847,000081:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
043848,000082:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
043849,000083:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
043850,000084:                                                                                                  #     EXECUTION.
043851,000085:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
043852,000086:                                                                                                  #     SUBLISTS.
043853,000087:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
043854,000088:                                                                                                  #     DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
043855,000089:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
043856,000090:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
043857,000091:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
043858,000092:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
043859,000093:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
043860,000094:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
043861,000095:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
043862,000096:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
043863,000097:                                                                                                  #     A. 1DNADR              1-WORD DOWNLIST ADDRESS.
043864,000098:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
043865,000099:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
043866,000100:                                                                                                  #     B. 2DNADR - 6DNADR     N-WORD DOWNLIST ADDRESS, N = 2 - 6.
043867,000101:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
043868,000102:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
043869,000103:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
043870,000104:                                                                                                  #     C. DNCHAN              DOWNLIST CHANNEL ADDRESS.
043871,000105:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
043872,000106:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
043873,000107:                                                                                                  #     D. DNPTR               DOWN TELEMETRY SUBLIST POINTER.
043874,000108:                                                                                                  #      SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
043875,000109:                                                                                                  #      CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
043876,000110:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
043877,000111:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST(ONLY).
043878,000112:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

044026,000260:                                                                                                  #  THE FOLLOWING CODING (FROM SNAPLOOP TO SNAPEND)IS FOR THE PURPOSE OF TAKING A SNAPSHOT OF 12 DP REGISTERS.
044027,000261:                                                                                                  #  THIS IS DONE BY SAVING 11 DP REGISTERS IN DNTMBUFF AND SENDING THE FIRST DP WORD IMMEDIATELY.

044105,000339:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL 8 BANKS OF ERASABLE STORAGE TWICE.  BANKS ARE SENT ONE AT A TIME
044106,000340:                                                                                                  #           EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
044107,000341:                                                                                                  #           EBANK. EBANKS ARE DUMPED IN ORDER(I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
044108,000342:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
044109,000343:                                                                                                  #           AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
044110,000344:                                                                                                  #           WILL BEGIN.
044111,000345:                                                                                                  #           ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
