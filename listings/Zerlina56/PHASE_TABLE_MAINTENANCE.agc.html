These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Linden Sims. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

056240,000074:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
056241,000075:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX   CONTAINS THE PHASE INFORMATION,
056242,000076: 
056243,000077:                                                                                                  #                                                   TC     PHASCHNG
056244,000078:                                                                                                  #                                                   OCT    XXXXX
056245,000079: 
056246,000080:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATIONTHAT IS STORED PERMANENTLY.  THESE
056247,000081:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
056248,000082: 
056249,000083:                                                                                                  #                  G.0             INACTIVE, WILLNOT PERMIT A GROUP G RESTART
056250,000084:                                                                                                  #                  G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
056251,000085:                                                                                                  #                  G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
056252,000086:                                                                                                  #                                  LONGCALL TO BE RESTARTED.
056253,000087:                                                                                                  #                  G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
056254,000088: 
056255,000089:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
056256,000090: 
056257,000091:                                                                                                  #                  TL0 00P PPP PPP GGG                                     ,
056258,000092: 
056259,000093:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7, THE P:S FOR THE PHASE,
056260,000094:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
056261,000095:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
056262,000096:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
056263,000097: 
056264,000098:                                                                                                  #                                                   TC     PHASCHNG        THIS WILL CAUSE GROUP 3 TOBE SET TO 0,
056265,000099:                                                                                                  #                                                   OCT    00003           MAKING GROUP 3 INACTIVE
056266,000100: 
056267,000101:                                                                                                  #                                                   TC     PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
056268,000102:                                                                                                  #                                                   OCT    00012           GROUP 2 TO RESTART THE LAST DISPLAY
056269,000103: 
056270,000104:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
056271,000105:                                                                                                  #                                                   OCT    40064           CASE OF A RESTART WOULD START UP THE TWO
056272,000106:                                                                                                  #                                                                          THINGS LOCATED IN THE DOUBLE 4.6 RESTART
056273,000107:                                                                                                  #                                                                          LOCATION
056274,000108:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
056275,000109:                                                                                                  #                                                   OCT    20135           CAUSES 5.13 TO BE RESTARTED  (SINCE
056276,000110:                                                                                                  #                                                                          LONGBASE WAS SET THIS SINGLE ENTRY
056277,000111:                                                                                                  #                                                                          SHOULD BE A LONGCALL)

056282,000116:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION. INSTEAD OF THE INFORMATION BEING IN A
056283,000117:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION. THE BITS ARE AS FOLLOWS,
056284,000118: 
056285,000119:                                                                                                  #                  TL0 1AD XXX CJW GGG
056286,000120:
