Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

042752,000041:                                                                                                  #  NAME....TVCRESTART PACKAGE, CONSISTING OF REDOTVC, ENABL1, 2, CMDSOUT, PHSCHK2, ETC.
042753,000042:                                                                                                  #  LOG SECTION...TVCRESTARTS                     SUBROUTINE...DAPCSM
042754,000043:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042755,000044: 
042756,000045:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042757,000046: 
042758,000047:                                                                                                  #       *RESTART-PROOFS THE TVC DAPS, INCLUDING PITCHDAP, YAWDAP,
042759,000048:                                                                                                  #        TVCEXECUTIVE, ROLLDAP, TVCINIT4, TVCDAPON, AND CSM/.M V46 SWTCHOVR.
042760,000049: 
042761,000050:                                                                                                  #       *TVC RESTARTS DESERVE SPECIAL CONSIDERATION IN SEVERAL AREAS.
042762,000051:                                                                                                  #        RESTART DOWN-TIME IS IMPORTANT BECAUSE OF THE TRANSIENTS INTRODUCED
042763,000052:                                                                                                  #        BY THE THRUST VECTOR RETURN TO THE ACTUATOR MECHANICAL NULLS
042764,000053:                                                                                                  #        FOLLOWING TVC- AND OPTICS-ERROR-COUNTER-DISENABLES (CHANNEL 12).
042765,000054:                                                                                                  #        TVC USES A MIXTURE OF WAITLIST, T5, T6, AND JOB CALLS.  THERE IS
042766,000055:                                                                                                  #        FILTER MEMORY (UP TO 6TH ORDER) TO BE PROTECTED IF WILD TRANSIENTS
042767,000056:                                                                                                  #        ARE TO BE AVOIDED.  COUNTERS ARE INVOLVED FOR ONE-SHOT
042768,000057:                                                                                                  #        CORRECTIONS AND GAIN UPDATES.  THE GIMBAL TRIM ESTIMATORS AND THE
042769,000058:                                                                                                  #        BODY AXIS ATTITUDE ERROR INTEGRATORS INVOLVE DIGITAL SUMMATION.
042770,000059:                                                                                                  #        DIGITAL DIFFERENTIATORS ARE INVOLVED IN THE BODY AXIS RATE ESTIMA-
042771,000060:                                                                                                  #        TIONS AND IN THE OUTPUTTING OF ACTUATOR COMMANDS.  THERE IS AN
042772,000061:                                                                                                  #        OFFSET-TRACKER-FILTER TO PROTECT.  ETC., ETC.
042773,000062: 
042774,000063:                                                                                                  #       *THOSE QUANTITIES WHICH MUST BE PROTECTED ARE STORED IN TEMPORARY
042775,000064:                                                                                                  #        REGISTERS AS THEY ARE COMPUTED, FOR UPDATING THE REAL REGISTERS
042776,000065:                                                                                                  #        DURING COPYCYCLES.
042777,000066: 
042778,000067:                                                                                                  #       *THE SEVERAL COPYCYCLES ARE EACH PROTECTED BY PHASE POINTS AT THEIR
042779,000068:                                                                                                  #        BEGINNING AND AT THEIR TERMINATION.  THE PHASE POINTS ARE SIMPLY
042780,000069:                                                                                                  #        ..INCR.. INSTRUCTIONS, EITHER ..INCR TVCEXPHS.. FOR COPYCYCLES
042781,000070:                                                                                                  #        IN THE TVCEXECUTIVE, OR ..INCR TVCPHASE.. FOR THE PITCH AND YAW
042782,000071:                                                                                                  #        COPYCYCLES.  INDEXING ON EACH OF THESE POINTERS THEN PERMITS A
042783,000072:                                                                                                  #        RETURN TO THE APPROPRIATE RESTART POINTS.
042784,000073: 
042785,000074:                                                                                                  #       *IF A RESTART OCCURS DURING EITHER COPYCYCLE, THAT COPYCYCLE IS
042786,000075:                                                                                                  #        COMPLETED.  THEN THE NORMAL TVCINIT4....DAPINIT....PITCHDAP STARTUP
042787,000076:                                                                                                  #        SEQUENCE IS CALLED UPON TO GET THINGS GOING AGAIN.
042788,000077: 
042789,000078:                                                                                                  #       *TVC-ENABLE AND OPTICS-ERROR-COUNTER ENABLE MUST BE SET ASAP
042790,000079:                                                                                                  #        (ALLOWING FOR PROCEDURAL DELAYS).  THEN THE ENGINES ARE COMMANDED
042791,000080:                                                                                                  #        TO THE P,YACTOFF TRIM VALUES.  THE DAPS ARE THEN READY TO GO ON THE
042792,000081:                                                                                                  #        AIR, WITH THE REGULAR STARTUP SEQUENCE, EITHER AT MRCLEAN FOR A
042793,000082:                                                                                                  #        COMPLETE INITIALIZATION OR AT TVCINIT4 FOR A PARTIAL INITIALIZATION
042794,000083: 
042795,000084:                                                                                                  #       *FOR RESTARTS PRIOR TO THE SETTING OF THE T5 BITS AT DOTVCON THE
042796,000085:                                                                                                  #        PRE40.6 SECTION OF S40.6 TAKES CARE OF RE-ESTABLISHING TRIMS.
042797,000086: 
042798,000087:                                                                                                  #       *IF A RESTART OCCURS DURING THE TVCEXEC....TVCEXFIN SEQUENCE THE
042799,000088:                                                                                                  #        COMPUTATIONS WILL BE COMPLETED, STARTING AT THE APPROPRIATE RESTART
042800,000089:                                                                                                  #        POINT, AFTER THE DAPS ARE READY TO GO ON THE AIR.
042801,000090: 
042802,000091:                                                                                                  #       *IF A RESTART OCCURS PRIOR TO TVCINIT4 (TVCPHASE = -1) E.G. DURING
042803,000092:                                                                                                  #        THE EARLY DAP INITIALIZATION PHASE, THE DAP STARTUP SEQUENCE IS
042804,000093:                                                                                                  #        ENTERED AT MRCLEAN FOR A FULL INITIALIZATION.
042805,000094: 
042806,000095:                                                                                                  #       *FOR RESTARTS DURING CSM/LM V46 SWITCH-OVER, TVCPHASE IS SET TO -2.
042807,000096:                                                                                                  #        AND THE RESTART LOGIC GOES BACK TO REDO SWITCH-OVER (AFTER THE
042808,000097:                                                                                                  #        NORMAL DAP RESTART SEQUENCE IS FOLLOWED).
042809,000098: 
042810,000099:                                                                                                  #       *RESTARTS ARE NOT CRITICAL TO THE ROLL DAP PERFORMANCES HENCE THE
042811,000100:                                                                                                  #        THE ROLL DAP IS MERELY RESTARTED.
042812,000101: 
042813,000102:                                                                                                  #       *RESTARTS DURING A STROKE TEST (STROKER IS NON-ZERO) WILL CAUSE THE

Page 957 

042815,000104:                                                                                                  #        STROKE TEST TO BE TERMINATED.  A NEW V68 ENTRY WILL BE REQUIRED
042816,000105:                                                                                                  #        TO GET IT GOING AGAIN (NO AUTOMATIC RESTART).
042817,000106:                                                                                                  #   
042818,000107:                                                                                                  #       *REDOTVC IS REACHED FOLLOWING ANY RESTART WHICH FINDS THE T5 BITS
042819,000108:                                                                                                  #        (BITS 15,14 OF FLAGWRD6) SET FOR TVC.  DOTVCON SETS TVCPHASE = -1
042820,000109:                                                                                                  #        AND TVC EXPHS = 0 JUST BEFORE SETTING THESE BITS, JUST BEFORE
042821,000110:                                                                                                  #        MAKING THE T5 CALL TO TVCDAPON.  ON A NORMAL SHUTDOWN DOTVCRCS
042822,000111:                                                                                                  #        CALLS RCSDAPON, WHICH RESETS THE T5 BITS FOR RCS.
042823,000112: 
042824,000113:                                                                                                  #  CALLING SEQUENCE....T5, IN PARTICULAR BY ELRSKIP OF FRESH START/RESTART
042825,000114: 
042826,000115:                                                                                                  #  NORMAL EXIT MODES....RESUME, NOQRSM, POSTJUMP (TO TVCINIT4 OR MRCLEAN)
042827,000116: 
042828,000117:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042829,000118: 
042830,000119:                                                                                                  #  SUBROUTINES CALLED....
042831,000120: 
042832,000121:                                                                                                  #       *PCOPY+1, YCOPY+1 (PITCH AND YAW COPYCYCLES)
042833,000122:                                                                                                  #       *ENABLE1,2, CMDSOUT (RE-ESTABLISH ACTUATOR TRIMS)
042834,000123:                                                                                                  #       *MRCLEAN OR TVCINIT4 (TVCDAP INITIALIZATIONS)
042835,000124:                                                                                                  #       *SWICHOVR +5 (CSM/LM V46 SWITCH-OVER)
042836,000125:                                                                                                  #       *EXRSTRT AND TVCEXECUTIVE PHASE POINTS 1 THRU 6
042837,000126:                                                                                                  #       *WAITLIST, IBNKCALL, POSTJUMP, ISWCALL
042838,000127: 
042839,000128:                                                                                                  #  OTHER INTERFACES....DOTVCON AND RCSDAPON (T5 BITS), ELRSKIP (CALLS IT)
042840,000129: 
042841,000130:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
042842,000131: 
042843,000132:                                                                                                  #       *T5 BITS (1,0), TVCPHASE (-2,-1,0,1,2,3), TVCEXPHS (1 THRU 6)
042844,000133:                                                                                                  #       *TVC DAP VARIABLES
042845,000134:                                                                                                  #       *OPERATIONS PERFORMED BY REDOTVC ARE BASED ON THE ASSUMPTION THAT
042846,000135:                                                                                                  #        THE TVC DAPS ARE RUNNING NORMALLY
042847,000136: 
042848,000137:                                                                                                  #  OUTPUT....
042849,000138: 
042850,000139:                                                                                                  #       *PITCH AND YAW TVC DAP COPYCYCLES COMPLETED IF INTERUPTED
042851,000140:                                                                                                  #       *TVCEXECUTIVE COMPLETED IF INTERUPTED
042852,000141:                                                                                                  #       *STROKE TEST TERMINATED IF INTERRUPTED
042853,000142:                                                                                                  #       *CSM/LM V46 SWITCH-OVER REPEATED IF INTERRUPTED
042854,000143:                                                                                                  #       *ACTUATOR TRIMS RE-ESTABLISHED (ACTUATORS BACK ON THE AIR)
042855,000144:                                                                                                  #       *TVC DAP INITIALIZATION AS REQUIRED
042856,000145:                                                                                                  #       *ALL TVC DAP OPERATIONS ON THE AIR
042857,000146: 
042858,000147:                                                                                                  #  DEBRIS....TVC TEMPORARIES IN EBANK6
