Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

053271,000047:                                                                                                  #  FUNCTIONAL DESCRIPTION -
053272,000048:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
053273,000049:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
053274,000050:                                                                                                  #        FOLLOW.
053275,000051: 
053276,000052:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
053277,000053: 
053278,000054:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
053279,000055:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
053280,000056:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
053281,000057:                                                                                                  #                                        .
053282,000058:                                                                                                  #                                        .
053283,000059:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
053284,000060:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
053285,000061: 
053286,000062:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
053287,000063:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
053288,000064:                                                                                                  #                                        .
053289,000065:                                                                                                  #                                        .
053290,000066:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
053291,000067:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
053292,000068: 
053293,000069:                                                                                                  #  WARNINGS-
053294,000070:                                                                                                  #  --------
053295,000071:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
053296,000072:                                                                                                  #        2)      9 TASKS MAXIMUM
053297,000073:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
053298,000074:                                                                                                  #        4)      TASKS END BY TC TASKOVER
053299,000075: 
053300,000076:                                                                                                  #  CALLING SEQUENCE -
053301,000077:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
053302,000078:                                                                                                  #        L       TC      WAITLIST
053303,000079:                                                                                                  #        L+1     2CADR   DESIRED TASK
053304,000080:                                                                                                  #        L+2     (MINOR OF 2CADR)
053305,000081:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
053306,000082: 
053307,000083:                                                                                                  #  TWIDDLE-
053308,000084:                                                                                                  #  -------
053309,000085:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
053310,000086:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

053312,000088:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
053313,000089:                                                                                                  #                L-1     CA      DELTAT
053314,000090:                                                                                                  #                L       TC      TWIDDLE
053315,000091:                                                                                                  #                L+1     ADRES   DESIRED TASK
053316,000092:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
053317,000093: 
053318,000094:                                                                                                  #  NORMAL EXIT MODES -
053319,000095:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
053320,000096: 
053321,000097:                                                                                                  #  ALARM OR ABORT EXIT MODES -
053322,000098:                                                                                                  #        TC      ABORT
053323,000099:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
053324,000100: 
053325,000101:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
053326,000102:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
053327,000103:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
053328,000104: 
053329,000105:                                                                                                  #  OUTPUT --
053330,000106:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
053331,000107: 
053332,000108:                                                                                                  #  DEBRIS -
053333,000109:                                                                                                  #        CENTRALS - A,Q,L
053334,000110:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
053335,000111: 
053336,000112:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
053337,000113:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
053338,000114:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
053339,000115:                                                                                                  #                LET TO = THE TIME OF THE TC WAITLIST
053340,000116:                                                                                                  #                LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
053341,000117:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
053342,000118:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
053343,000119:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
053344,000120:                                                                                                  #                (Z=0, USUALLY)
053345,000121:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
053346,000122:                                                                                                  #        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
053347,000123:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
053348,000124:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
053349,000125:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
053350,000126:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
