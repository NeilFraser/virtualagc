Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

060717,000035:                                                                                                  #  PROGRAM DESCRIPTION                           DATE  20 DECEMBER 1967
060718,000036:                                                                                                  #  PROGRAM NAME - SELF-CHECK                     LOG SECTION AGC BLOCK TWO SELF-CHECK
060719,000037:                                                                                                  #  MOD NO - 1                                    ASSEMBLY SUBROUTINE UTILITYM REV 25
060720,000038:                                                                                                  #  MOD BY - GAUNTT
060721,000039: 
060722,000040:                                                                                                  #  FUNCTIONAL DESCRIPTION
060723,000041:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
060724,000042:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
060725,000043:                                                                                                  #        STARTING VERB.
060726,000044: 
060727,000045:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
060728,000046: 
060729,000047:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
060730,000048: 
060731,000049:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
060732,000050:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
060733,000051: 
060734,000052:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
060735,000053:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
060736,000054:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
060737,000055:                                                                                                  #                +-4             ERASABLE MEMORY
060738,000056:                                                                                                  #                +-5             FIXED MEMORY
060739,000057:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
060740,000058:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
060741,000059:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
060742,000060: 
060743,000061:                                                                                                  #  WARNINGS
060744,000062:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
060745,000063:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
060746,000064: 
060747,000065:                                                                                                  #  CALLING SEQUENCE
060748,000066:                                                                                                  #        TO CALL SELF-CHECK KEY IN
060749,000067:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
060750,000068:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
060751,000069:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
060752,000070:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
060753,000071: 
060754,000072:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
060755,000073:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
060756,000074:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
060757,000075: 
060758,000076:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
060759,000077: 
060760,000078:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
060761,000079: 
060762,000080:                                                                                                  #  OUTPUT

Page 1395 

060764,000082:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
060765,000083:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
060766,000084:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
060767,000085:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
060768,000086:                                                                                                  #        INITIATED FRESH START (SLAP1).
060769,000087: 
060770,000088:                                                                                                  #        SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
060771,000089:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
060772,000090: 
060773,000091:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
060774,000092:                                                                                                  #        ACCOMPLISHED BY FRESH START
060775,000093:                                                                                                  #                SMODE SET TO +0
060776,000094: 
060777,000095:                                                                                                  #  DEBRIS
060778,000096:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
060779,000097:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
060780,000098:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
