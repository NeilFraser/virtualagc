Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

052167,000056:                                                                                                  #  FUNCTIONAL DESCRIPTION -
052168,000057:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
052169,000058:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
052170,000059:                                                                                                  #        FOLLOW.
052171,000060: 
052172,000061:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
052173,000062: 
052174,000063:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
052175,000064:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
052176,000065:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
052177,000066:                                                                                                  #                                        .
052178,000067:                                                                                                  #                                        .
052179,000068:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
052180,000069:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
052181,000070: 
052182,000071:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
052183,000072:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
052184,000073:                                                                                                  #                                        .
052185,000074:                                                                                                  #                                        .
052186,000075:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
052187,000076:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
052188,000077: 
052189,000078:                                                                                                  #  WARNINGS-
052190,000079:                                                                                                  #  --------
052191,000080:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
052192,000081:                                                                                                  #        2)      9 TASKS MAXIMUM
052193,000082:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
052194,000083:                                                                                                  #        4)      TASKS END BY TC TASKOVER
052195,000084: 
052196,000085:                                                                                                  #  CALLING SEQUENCE -
052197,000086:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
052198,000087:                                                                                                  #        L       TC      WAITLIST
052199,000088:                                                                                                  #        L+1     2CADR   DESIRED TASK
052200,000089:                                                                                                  #        L+2     (MINOR OF 2CADR)
052201,000090:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
052202,000091: 
052203,000092:                                                                                                  #  TWIDDLE-
052204,000093:                                                                                                  #  -------
052205,000094:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
052206,000095:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

052208,000097:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
052209,000098:                                                                                                  #                L-1     CA      DELTAT
052210,000099:                                                                                                  #                L       TC      TWIDDLE
052211,000100:                                                                                                  #                L+1     ADRES   DESIRED TASK
052212,000101:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
052213,000102: 
052214,000103:                                                                                                  #  NORMAL EXIT MODES -
052215,000104:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
052216,000105: 
052217,000106:                                                                                                  #  ALARM OR ABORT EXIT MODES -
052218,000107:                                                                                                  #        TC      ABORT
052219,000108:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
052220,000109: 
052221,000110:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
052222,000111:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
052223,000112:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
052224,000113: 
052225,000114:                                                                                                  #  OUTPUT --
052226,000115:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
052227,000116: 
052228,000117:                                                                                                  #  DEBRIS -
052229,000118:                                                                                                  #        CENTRALS - A,Q,L
052230,000119:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
052231,000120: 
052232,000121:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
052233,000122:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
052234,000123:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
052235,000124:                                                                                                  #                LET TO = THE TIME OF THE TC WAITLIST
052236,000125:                                                                                                  #                LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
052237,000126:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
052238,000127:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
052239,000128:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
052240,000129:                                                                                                  #                         (Z=0, USUALLY)
052241,000130:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
052242,000131:                                                                                                  #                        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
052243,000132:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
052244,000133:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
052245,000134:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
052246,000135:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
