Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

047447,000047:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
047448,000048:                                                                                                  #  MOD NO.- 0    TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
047449,000049:                                                                                                  #                PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
047450,000050:                                                                                                  #                THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.  REFER TO DOWNLINK
047451,000051:                                                                                                  #                LISTS LOG SECTION FOR MORE DETAILS.  HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
047452,000052:                                                                                                  #                OF DOWN TELEMETRY DATA.
047453,000053:                                                                                                  #  MOD BY-       KILROY, SMITH, DEWITT
047454,000054:                                                                                                  #  DATE-         02 OCT 67
047455,000055:                                                                                                  #  AUTHORS-      KILROY, SMITH, DEWITT, DEWOLF, FAGIN
047456,000056:                                                                                                  #  LOG SECTION-  DOWN-TELEMETRY PROGRAM
047457,000057: 
047458,000058:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
047459,000059:                                                                                                  #        PULSE FROM THE DOWNLINK TELEMETRY CONVERTER.  THIS PULSE OCCURS
047460,000060:                                                                                                  #        AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
047461,000061:                                                                                                  #        EXECUTED AT THESE RATES.  THIS ROUTINE SELECTS THE APPROPRIATE
047462,000062:                                                                                                  #        AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
047463,000063:                                                                                                  #        CHANNELS 34 AND 35.  THE INFORMATION IS THEN GATED OUT FROM THE
047464,000064:                                                                                                  #        LGC IN SERIAL FASHION.
047465,000065: 
047466,000066:                                                                                                  #        THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST.  SINCE DOWNRUPTS
047467,000067:                                                                                                  #        OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
047468,000068:                                                                                                  #        CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
047469,000069:                                                                                                  #        OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
047470,000070: 
047471,000071:                                                                                                  #  CALLING SEQUENCE- NONE
047472,000072:                                                                                                  #        PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
047473,000073:                                                                                                  #        RESULT OF A DOWNRUPT.  CONTROL IS RETURNED VIA TCF RESUME WHICH
047474,000074:                                                                                                  #        IN EFFECT IS A RESUME.
047475,000075: 
047476,000076:                                                                                                  #  SUBROUTINES CALLED- NONE
047477,000077: 
047478,000078:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
047479,000079: 
047480,000080:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
047481,000081: 
047482,000082:                                                                                                  #  RESTART PROTECTION:
047483,000083:                                                                                                  #        ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
047484,000084:                                                                                                  #        DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E. CURRENT CONTENTS OF DNLSTADR).  THIS HAS THE
047485,000085:                                                                                                  #        EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
047486,000086:                                                                                                  #        THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
047487,000087:                                                                                                  #        DOWNLIST.
047488,000088: 
047489,000089:                                                                                                  #        ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
047490,000090: 
047491,000091:                                                                                                  #  DOWNLINK LIST SELECTION:
047492,000092:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
047493,000093:                                                                                                  #        1.      FRESH START
047494,000094:                                                                                                  #        2.      V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
047495,000095:                                                                                                  #        3.      UPDATE PROGRAM (P27)
047496,000096:                                                                                                  #        4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
047497,000097:                                                                                                  #                TRANSITION (COLOSSUS) ETC.).
047498,000098: 
047499,000099:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
047500,000100:                                                                                                  #        READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
047501,000101: 
047502,000102:                                                                                                  #  OUTPUT- EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
047503,000103:                                                                                                  #        COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
047504,000104: 
047505,000105:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
047506,000106:                                                                                                  #        'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
047507,000107: 
047508,000108:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
047509,000109:                                                                                                  #        LDATALST, DNTMBUFF TO DNTMBUFF +21D, TMINDEX, DNQ.

047514,000114:                                                                                                  #  DODOWNTM IS ENTERED EVERY 20 MS BY AN INTERRUPT TRIGGERED BY THE
047515,000115:                                                                                                  #  RECEIPT OF AN ENDPULSE FROM THE SPACECRAFT TELEMETRY PROGRAMMER.
047516,000116: 
047517,000117:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
047518,000118:                                                                                                  #  1.    DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
047519,000119:                                                                                                  #        DOWN-TELEMETRY PROGRAM.  THIS IS DONE FOR EASE OF CODING, FASTER
047520,000120:                                                                                                  #        EXECUTION.
047521,000121:                                                                                                  #  2.    EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
047522,000122:                                                                                                  #        SUBLISTS.
047523,000123:                                                                                                  #  3.    A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
047524,000124:                                                                                                  #        DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
047525,000125:                                                                                                  #        CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
047526,000126:                                                                                                  #  4.    SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
047527,000127:                                                                                                  #        SAVED IN A BUFFER DURING ONE DOWNRUPT.
047528,000128:                                                                                                  #  5.    THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
047529,000129:                                                                                                  #        OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
047530,000130:                                                                                                  #        IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
047531,000131:                                                                                                  #  6.    ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
047532,000132:                                                                                                  #        CREATED FOR THE DOWNLIST PROGRAM.
047533,000133:                                                                                                  #        A.      1DNADR                  1-WORD DOWNLIST ADDRESS.
047534,000134:                                                                                                  #                SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
047535,000135:                                                                                                  #                HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
047536,000136:                                                                                                  #        B.      2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
047537,000137:                                                                                                  #                SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
047538,000138:                                                                                                  #                FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
047539,000139:                                                                                                  #                PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
047540,000140:                                                                                                  #        C.      DNCHAN                  DOWNLIST CHANNEL ADDRESS.
047541,000141:                                                                                                  #                SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
047542,000142:                                                                                                  #                A PAIR OF CHANNELS FOR DOWN TELEMETRY.
047543,000143:                                                                                                  #        D.      DNPTR                   DOWN TELEMETRY SUBLIST POINTER.
047544,000144:                                                                                                  #                SAME AS CAF BUT TAGGED AS A CONSTANT.  USED IN CONTROL LIST TO POINT TO A SUBLIST.
047545,000145:                                                                                                  #                CAUTION --- A DNPTR CANNOT BE USED IN A SUBLIST.
047546,000146:                                                                                                  #  7.    THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
047547,000147:                                                                                                  #        A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
047548,000148:                                                                                                  #  8.    IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.
047549,000149: 
047550,000150:                                                                                                  #  DOWNLINK LIST RESTRICTIONS:
047551,000151:                                                                                                  #  (THE FOLLOWING POINTS MAY BE LISTED ELSEWHERE BUT ARE LISTED HERE SO IT IS CLEAR THAT THESE THINGS CANNOT BE
047552,000152:                                                                                                  #  DONE)
047553,000153:                                                                                                  #  1.    SNAPSHOT DOWNLIST:
047554,000154:                                                                                                  #        (A) CANNOT CONTAIN THE FOLLOWING ECADRS (I.E. 1DNADR'S): 0, 400, 1000, 1400, 2000, 2400, 3000, 3400.
047555,000155:                                                                                                  #        (B) CAN CONTAIN ONLY 1DNADR'S
047556,000156:                                                                                                  #  2.    ALL DOWNLINKED DATA (EXCEPT CHANNELS) IS PICKED UP BY A <DCA<SO DOWNLINK LISTS CANNOT CONTAIN THE
047557,000157:                                                                                                  #        EQUIVALENT OF THE FOLLOWING ECADRS (I.E. IDNADRS): 377, 777, 1377, 1777, 2377, 27777, 3377, 3777.
047558,000158:                                                                                                  #        (NOTE: THE TERM EQUIVALENT ' MEANT THAT THE IDNADR TO 6DNADR WILL BE PROCESSED LIKE 1 TO 6 ECADRS)
047559,000159:                                                                                                  #  3.    CONTROL LISTS AND SUBLISTS CANNOT HAVE ENTRIES = OCTAL 00000 OR OCTAL 77777
