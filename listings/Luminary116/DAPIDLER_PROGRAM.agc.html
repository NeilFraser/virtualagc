Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

059887,000339: 
059888,000340:                                                                                                  #  PROGRAM DESCRIPTION:    D. KEENE   5/24/67
059889,000341:                                                                                                  #  MOD 1 BY CRAIG WORK, 12 DEC 67
059890,000342:                                                                                                  #  MOD 2 BY CRAIG WORK, 6 APRIL 68 CONVERTS ATTITUDE ERROR DISPLAY SCALING FROM 16 7/8 DEG. TO 42 3/16 DEGREES.
059891,000343: 
059892,000344:                                                                                                  #      THIS SUBROUTINE IS USED TO DISPLAY ATTITUDE ERRORS ON THE FDAI VIA THE DIGITAL TO ANALOG CONVERTERS (DACS)
059893,000345:                                                                                                  #  IN THE CDUS.  CARE IS TAKEN TO METER OUT THE APPROPRIATE NUMBER OF PULSES TO THE IMU ERROR COUNTERS AND PREVENT
059894,000346:                                                                                                  #  OVERFLOW, TO CONTROL THE RELAY SEQUENCING, AND TO AVOID INTERFERENCE WITH THE COARSE ALIGN LOOP WHICH ALSO USES
059895,000347:                                                                                                  #  THE DACS.
059896,000348: 
059897,000349:                                                                                                  #  CALLING SEQUENCE:
059898,000350:                                                                                                  #      DURING THE INITIALIZATION SECTION OF THE USER'S PROGRAM, BIT3 OF RCSFLAGS SHOULD BE SET TO INITIATE THE
059899,000351:                                                                                                  #  TURN-ON SEQUENCE WITHIN THE NEEDLES PROGRAM:
059900,000352:                                                                                                  #           CS     RCSFLAGS        IN EBANK6
059901,000353:                                                                                                  #           MASK   BIT3
059902,000354:                                                                                                  #           ADS    RCSFLAGS
059903,000355:                                                                                                  #  THEREAFTER, THE ATTITUDE ERRORS GENERATED BY THE USER SHOULD BE TRANSFERED TO THE FOLLOWING LOCATIONS IN EBANK6:
059904,000356:                                                                                                  #           AK     SCALED 180 DEGREES  NOTE: THESE LOCATIONS ARE SUBJECT
059905,000357:                                                                                                  #           AK1    SCALED 180 DEGREES        TO CHANGE
059906,000358:                                                                                                  #           AK2    SCALED 180 DEGREES
059907,000359:                                                                                                  #  FULL SCALED DEFLECTION OF THE NEEDLES CORRESPONDS TO 5 1/16 DEGREES, WHILE 384 BITS IN THE IMU ERROR COUNTER
059908,000360:                                                                                                  #  CORRESPONDS TO 42 3/16 DEGREES.  (DAC MAXIMUM CAPACITY IS 384 BITS.) 46 BITS EFFECTIVELY PIN THE NEEDLES.
059909,000361: 
059910,000362:                                                                                                  #  A CALL TO NEEDLER WILL THEN UPDATE THE DISPLAY:
059911,000363:                                                                                                  #           INHINT

059913,000365:                                                                                                  #           CADR   NEEDLER
059914,000366:                                                                                                  #           RELINT

059916,000368:                                                                                                  #  REQUIRED BEFORE ANYTHING IS ACTUALLY DISPLAYED ON THE ERROR METERS.
