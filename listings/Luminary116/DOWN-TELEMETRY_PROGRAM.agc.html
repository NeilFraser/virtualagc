Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

042323,000026:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
042324,000027:                                                                                                  #  MOD NO.- 0      TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
042325,000028:                                                                                                  #                  PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
042326,000029:                                                                                                  #                  THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.REFER TO DOWNLINK
042327,000030:                                                                                                  #                  LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
042328,000031:                                                                                                  #                  OF DOWN TELEMETRY DATA.
042329,000032:                                                                                                  #  MOD BY-  KILROY, SMITH, DEWITT
042330,000033:                                                                                                  #  DATE-    02 OCT 67
042331,000034:                                                                                                  #  AUTHORS- KILROY, SMITH, DEWITT, DEWOLF, FAGIN
042332,000035:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
042333,000036:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
042334,000037:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
042335,000038:                                                                                                  #           AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
042336,000039:                                                                                                  #           EXECUTED AT THESE RATES.  THIS ROUTINE SELECTS THE APPROPRIATE
042337,000040:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
042338,000041:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
042339,000042:                                                                                                  #           LGC IN SERIAL FASHION.
042340,000043:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
042341,000044:                                                                                                  #           OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
042342,000045:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
042343,000046:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
042344,000047:                                                                                                  #  CALLING SEQUENCE- NONE
042345,000048:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
042346,000049:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
042347,000050:                                                                                                  #           IN EFFECT IS A RESUME.
042348,000051:                                                                                                  #  SUBROUTINES CALLED- NONE
042349,000052:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
042350,000053:                                                                                                  #  ALARM OR ABORT EXIT MODE - NONE
042351,000054:                                                                                                  #  RESTART PROTECTION:
042352,000055:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
042353,000056:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST(I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
042354,000057:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
042355,000058:                                                                                                  #           THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
042356,000059:                                                                                                  #           DOWNLIST.
042357,000060:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
042358,000061:                                                                                                  #  DOWNLINK LIST SELECTION:
042359,000062:                                                                                                  #           THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
042360,000063:                                                                                                  #           1.  FRESH START
042361,000064:                                                                                                  #           2.  V37EXXE  WHERE XX = THE MAJOR MODE BEING SELECTED.
042362,000065:                                                                                                  #           3.  UPDATE PROGRAM (P27)
042363,000066:                                                                                                  #           4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION(SUNDANCE, LUMINARY) AND P61-P62
042364,000067:                                                                                                  #                   TRANSITION(COLOSSUS) ETC.).
042365,000068: 
042366,000069:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
042367,000070:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
042368,000071: 
042369,000072:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
042370,000073:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
042371,000074: 
042372,000075:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
042373,000076:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
042374,000077: 
042375,000078:                                                                                                  #  DEBRIS(ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
042376,000079:                                                                                                  #        LDATALST, DNTMBUFF TO DNTMBUFF +21D, TMINDEX, DNQ.

042382,000085:                                                                                                  #  DODOWNTM IS ENTERED EVERY 20 MS BY AN INTERRUPT TRIGGERED BY THE
042383,000086:                                                                                                  #  RECEIPT OF AN ENDPULSE FROM THE SPACECRAFT TELEMETRY PROGRAMMER.
042384,000087: 
042385,000088:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
042386,000089:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
042387,000090:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
042388,000091:                                                                                                  #     EXECUTION.
042389,000092:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
042390,000093:                                                                                                  #     SUBLISTS.
042391,000094:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
042392,000095:                                                                                                  #     DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
042393,000096:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
042394,000097:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
042395,000098:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
042396,000099:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
042397,000100:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
042398,000101:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
042399,000102:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
042400,000103:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
042401,000104:                                                                                                  #     A. 1DNADR              1-WORD DOWNLIST ADDRESS.
042402,000105:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
042403,000106:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
042404,000107:                                                                                                  #     B. 2DNADR - 6DNADR     N-WORD DOWNLIST ADDRESS, N = 2 - 6.
042405,000108:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
042406,000109:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
042407,000110:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
042408,000111:                                                                                                  #     C. DNCHAN              DOWNLIST CHANNEL ADDRESS.
042409,000112:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
042410,000113:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
042411,000114:                                                                                                  #     D. DNPTR               DOWN TELEMETRY SUBLIST POINTER.
042412,000115:                                                                                                  #         SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
042413,000116:                                                                                                  #         CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
042414,000117:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
042415,000118:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
042416,000119:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.
042417,000120:

042419,000122:                                                                                                  #    (THE FOLLOWING POINTS MAY BE LISTED ELSEWHERE BUT ARE LISTED HERE SO IT IS CLEAR THAT THESE THINGS CANNOT BE
042420,000123:                                                                                                  #    DONE)

042422,000125:                                                                                                  #  1. SNAPSHOT DOWNLIST:
042423,000126:                                                                                                  #    (A) CANNOT CONTAIN THE FOLLOWING ECADRS(I.E. 1DNADR'S): 0, 400, 1000, 1400, 2000, 2400, 3000, 3400.
042424,000127:                                                                                                  #    (B) CAN CONTAIN ONLY 1DNADR'S

042426,000129:                                                                                                  #  2. ALL DOWNLINKED DATA(EXCEPT CHANNELS) IS PICKED UP BY A <DCA<SO DOWNLINK LISTS CANNOT CONTAIN THE
042427,000130:                                                                                                  #     EQUIVALENT OF THE FOLLOWING ECADRS(I.E. IDNADRS): 377, 777, 1377, 1777, 2377, 27777, 3377, 3777.
042428,000131: 
042429,000132:                                                                                                  #     (NOTE: THE TERM EQUIVALENT ' MEANT THAT THE IDNADR TO 6DNADR  WILL BE PROCESSED LIKE 1 TO 6 ECADRS)

042638,000341:                                                                                                  #  SUBROUTINE NAME- DNDUMP
042639,000342: 
042640,000343:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL 8 BANKS OF ERASABLE STORAGE TWICE.  BANKS ARE SENT ONE AT A TIME
042641,000344:                                                                                                  #           EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
042642,000345:                                                                                                  #           EBANK. EBANKS ARE DUMPED IN ORDER (I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
042643,000346: 
042644,000347:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
042645,000348:                                                                                                  #           AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
042646,000349:                                                                                                  #           WILL BEGIN.
042647,000350:                                                                                                  #           ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
042648,000351:                                                                                                  #           BY THE FOLLOWING:
042649,000352:                                                                                                  #            1. A FRESH START
042650,000353:                                                                                                  #            2. COMPLETION OF BOTH COMPLETE DUMPS
042651,000354:                                                                                                  #            3. AND INVOLUNTARILY BY A RESTART.
042652,000355:                                                                                                  #  NORMAL EXIT MODE- TCF DNPHASE1
042653,000356:                                                                                                  #  ALARM OR ABORT MODE- NONE
042654,000357:                                                                                                  #  *SUBROUTINES CALLED- NONE.
042655,000358:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
042656,000359:                                                                                                  #           NONE
042657,000360:                                                                                                  #  DEBRIS - DUMPLOC, DUMPSW, DNTMGOTO, EBANK AND CENTRAL REGISTERS
042658,000361:                                                                                                  #  TIMING -   TIME (IN SECS) = ((NO.DUMPS)*(NO.EBANKS)*(WDSPEREBANK + NO.IDWDS)) / NO.WDSPERSEC
042659,000362:                                                                                                  #             TIME (IN SECS) =  (   4    )*(    8    )*(    256     +     4   )  /     100
