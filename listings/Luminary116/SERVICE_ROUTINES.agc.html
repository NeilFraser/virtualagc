Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

058035,000050:                                                                                                  #      UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
058036,000051:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
058037,000052:                                                                                                  #  EBANK SETTING.   A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
058038,000053:                                                                                                  #  ASSEMBLER.
058039,000054: 
058040,000055:                                                                                                  #      AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
058041,000056:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
058042,000057: 
058043,000058:                                                                                                  #      CALLING SEQUENCES ARE AS FOLLOWS:-
058044,000059:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
058045,000060:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG

058153,000168:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
058154,000169: 
058155,000170:                                                                                                  #  WRITTEN BY D. EYLES
058156,000171:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
058157,000172: 
058158,000173:                                                                                                  #  MOD 2 BY SCHULENBERG (REMOVE RELINT) SKIPPER REV 4 2/28/68
058159,000174: 
058160,000175:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
058161,000176:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
058162,000177: 
058163,000178:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
058164,000179:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
058165,000180:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
058166,000181:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
058167,000182:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
058168,000183:                                                                                                  #        I +4                            RETURNS HERE.
058169,000184: 
058170,000185:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
058171,000186: 
058172,000187:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
058173,000188: 
058174,000189:    5544                                           BLOCK    02                                    
058175,000190:    4000                                           SETLOC   FFTAG4                                
058176,000191:    4000                                           BANK                                           
058177,000192: 
058178,000193:    5544     0061                                  EBANK=   ITEMP1                                
058179,000194: 
058180,000195:    5544                                           COUNT*   $$/TRAN                               
058181,000196: 
058182,000197:    5544           00004        GENTRAN            INHINT                                         
058183,000198:    5545           54061                           TS       ITEMP1                                #  SAVE N-1.
058184,000199:    5546           50002                           INDEX    Q                                     #  C(Q) = ADRES L.
058185,000200:    5547           60000                           AD       0                                     #  ADRES (L + N - 1).
058186,000201:    5550           50000                           INDEX    A                                     
058187,000202:    5551           30000                           CA       0                                     #  C(ABOVE).
058188,000203:    5552           54001                           TS       L                                     #  SAVE DATA.
058189,000204:    5553           30061                           CA       ITEMP1
