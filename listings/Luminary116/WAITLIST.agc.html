Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

046965,000051:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
046966,000052:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2

046968,000054:                                                                                                  #                                          .
046969,000055:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
046970,000056:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9

046973,000059:                                                                                                  #  --------

046975,000061:                                                                                                  #               2) 9 TASKS MAXIMUM

047023,000109:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
047024,000110:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
047025,000111: 
047026,000112:                                                                                                  #               LET TO = THE TIME OF THE TC WAITLIST
047027,000113:                                                                                                  #               LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
047028,000114:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
047029,000115:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
047030,000116:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
047031,000117:                                                                                                  #     (Z=0, USUALLY)
047032,000118:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
047033,000119:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
047034,000120:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
047035,000121: 
047036,000122: 
047037,000123:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
047038,000124:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
047039,000125:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
