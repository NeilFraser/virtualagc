018591,000033:                                                                                                  #  THIS LOG SECTION CONSISTS OF PRELAUNCH ALIGNMENT AND GYRO DRIFT TESTS
018592,000034:                                                                                                  #  INTEGRATED TOGETHER TO SAVE WORDS. IT IS COMPLETELY RESTART PROOFED.
018593,000035:                                                                                                  #  THE PRELAUNCH ALIGNMENT TECHNIQUE IS BASICALLY THE SAME AS IN BLOCK 1
018594,000036:                                                                                                  #  EXCEPT THAT IT HAS BEEN SIMPLIFIED IN THE SENSE THAT SMALL ANGLE APPROX.
018595,000037:                                                                                                  #  HAVE BEEN USED. THE DRIFT TESTS USE A UNIQUE IMPLEMENTATION OF THE
018596,000038:                                                                                                  #  OPTIMUM STATISTICAL FILTER. FOR A DESCRIPTION SEE E-1973.BOTH OF THESE
018597,000039:                                                                                                  #  ROUTINES USE STANDARD SYSTEM TEST LEADIN PROCEDURES. THE INITIALIZATION
018598,000040:                                                                                                  #  PROCEDURE FOR THE DRIFT TESTS IS IN THE JDC S. THE INITIALIZATION METHOD
018599,000041: 
018600,000042:                                                                                                  #  FOR GYROCOMPASS IS IN A SOON TO COME SYSTEM TEST GROUP MEMO.
018601,000043:                                                                                                  #  THIS LOG SECTION ALWAYS STARTS BY A CADR IN IMU PERF. TESTS 2 AND BEGINS
018602,000044:                                                                                                  #  AT ESTIMS. THE PIPS ARE READ EVERY .5 SEC IN COMPASS AND 1 SEC IN DRIFT
018603,000045:                                                                                                  #  TESTS. THEN IN BOTH CASES RELEVANT COMPUTATION IS DONE. THE KEY ERASABLE
018604,000046:                                                                                                  #  IS GEOCOMPS- 0 WE ARE IN A DRIFT TEST -NONZERO WE ARE IN COMPASS.
018605,000047:                                                                                                  #  THE GYROCOMPASS HAS THE CAPABILITY TO ALIGN TO ANY ORIENTATION,HAS THE
018606,000048:                                                                                                  #  CAPABILITY TO CHANGE ORIENTATIONS WHILE RUNNING,IS COMPENSATED FOR
018607,000049:                                                                                                  #  COMPONENT ERRORS,IS CAPABLE OF OPTICAL VERIFICATION( CSM ONLY).
018608,000050: 
018609,000051:                                                                                                  #  SUBROUTINES CALLED
018610,000052: 
018611,000053:                                                                                                  #  EARTHR, CALGTA,  OGC ZERO,ERTHRVSE,GCOMPZERO AND
018612,000054:                                                                                                  #  IMU COMPENSATION
018613,000055:                                                                                                  #  DURING OPTICAL VERIFICATION (CSM ONLY) ESSENTIALLY ALL OF INFLIGHT ALIGN
018614,000056:                                                                                                  #  IS CALLED IN ONE WAY OR ANOTHER. SEE THE LISTING.
