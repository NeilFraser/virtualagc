The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. 057524,000038:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES -
057525,000039:                                                                                                  #        1.  PRIORITY DISPLAYS - DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS.  USUALLY THESE DISPLAYS ARE SENT
057526,000040:                                                                                                  #            OUT UNDER CRITICAL ALARM CONDITIONS.
057527,000041:                                                                                                  #        2.  EXTENDED VERB DISPLAYS - ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
057528,000042:                                                                                                  #        3.  NORMAL DISPLAYS - ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
057529,000043:                                                                                                  #            SEQUENCE OF EVENTS.
057530,000044:                                                                                                  #        4.  MISC. DISPLAYS - ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES.  THESE INCLUDE SUCH DISPLAYS AS
057531,000045:                                                                                                  #            MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
057532,000046:                                                                                                  #        5.  ASTRONAUT INITIATED DISPLAYS - ALL DISPLAYS INITIATED EXTERNALLY.
057533,000047: 
057534,000048:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS -
057535,000049:                                                                                                  #        1.  ACTIVE - THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
057536,000050:                                                                                                  #            (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
057537,000051:                                                                                                  #            BUT NOT WAITING FOR A RESPONSE.
057538,000052:                                                                                                  #        2.  INACTIVE - A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
057539,000053:                                                                                                  #            (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
057540,000054:                                                                                                  #            WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
057541,000055:                                                                                                  #            OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
057542,000056:                                                                                                  #            HAS INFO SAVED FOR RESTART PURPOSES.
057543,000057: 
057544,000058:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS -
057545,000059:                                                                                                  #        INTERRUPTS -
057546,000060:                                                                                                  #                1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
057547,000061:                                                                                                  #                2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
057548,000062:                                                                                                  #                3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
057549,000063:                                                                                                  #                    PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
057550,000064:                                                                                                  #                4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
057551,000065:                                                                                                  #                5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
057552,000066: 
057553,000067:                                                                                                  #        ORDER OF WAITING DISPLAYS -
057554,000068:                                                                                                  #                1.  ASTRONAUT EXTERNAL USE
057555,000069:                                                                                                  #                2.  PRIORITY
057556,000070:                                                                                                  #                3.  INTERRUPTED MARK
057557,000071:                                                                                                  #                4.  INTERRUPTED NORMAL
057558,000072:                                                                                                  #                5.  MARK TO BE REQUESTED (SEE DESCRIPTION OF ENDMARK)
057559,000073:                                                                                                  #                6.  MARK WAITING
057560,000074:                                                                                                  #                7.  NORMAL WAITING

057563,000077:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
057564,000078:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
057565,000079:                                                                                                  #  WILL FOLLOW LATER):
057566,000080:                                                                                                  #        1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
057567,000081:                                                                                                  #            IS TO THE USERS CALL CADR +4.  FOR THE ONLY NON FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
057568,000082:                                                                                                  #            IS TO THE USERS CALLING LOC +1.
057569,000083:                                                                                                  #        2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.
057570,000084:                                                                                                  #        3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USERS JOB PRIORITY.
057571,000085:                                                                                                  #        4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
057572,000086:                                                                                                  #            USERS JOB.
057573,000087:                                                                                                  #        5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
057574,000088:                                                                                                  #        6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
057575,000089:                                                                                                  #            DESCRIPTION OF RESTARTS).
057576,000090:                                                                                                  #        7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPONSES.  A TERMINATE (V34) BRANCHES
057577,000091:                                                                                                  #            TO THE USERS CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USERS CALL CADR +2.  AN ENTER OR RECYCLE
057578,000092:                                                                                                  #            (V32) BRANCHES TO THE USERS CALL CADR +3.
057579,000093:                                                                                                  #        8.  ALL ROUTINES MUST BE USED UNDER EXECUTIVE CONTROL.
