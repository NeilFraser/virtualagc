Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	16:27 JULY 14,1969
Note that the date is the date of the printout, not the date of the program revision.

047797,000047:                                                                                                  #  FUNCTIONAL DESCRIPTION -
047798,000048:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
047799,000049:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
047800,000050:                                                                                                  #        FOLLOW.
047801,000051: 
047802,000052:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
047803,000053: 
047804,000054:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
047805,000055:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
047806,000056:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
047807,000057:                                                                                                  #                                        .
047808,000058:                                                                                                  #                                        .
047809,000059:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
047810,000060:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
047811,000061: 
047812,000062:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
047813,000063:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
047814,000064:                                                                                                  #                                        .
047815,000065:                                                                                                  #                                        .
047816,000066:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
047817,000067:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
047818,000068: 
047819,000069:                                                                                                  #  WARNINGS -
047820,000070:                                                                                                  #  --------
047821,000071:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
047822,000072:                                                                                                  #        2)      9 TASKS MAXIMUM
047823,000073:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
047824,000074:                                                                                                  #        4)      TASKS END BY TC TASKOVER
047825,000075: 
047826,000076:                                                                                                  #  CALLING SEQUENCE -
047827,000077:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
047828,000078:                                                                                                  #        L       TC      WAITLIST
047829,000079:                                                                                                  #        L+1     2CADR   DESIRED TASK
047830,000080:                                                                                                  #        L+2     (MINOR OF 2CADR)
047831,000081:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
047832,000082: 
047833,000083:                                                                                                  #  TWIDDLE -
047834,000084:                                                                                                  #  -------
047835,000085:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
047836,000086:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

Page 1118 

047838,000088:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
047839,000089:                                                                                                  #                L-1     CA      DELTAT
047840,000090:                                                                                                  #                L       TC      TWIDDLE
047841,000091:                                                                                                  #                L+1     ADRES   DESIRED TASK
047842,000092:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
047843,000093: 
047844,000094:                                                                                                  #  NORMAL EXIT MODES -
047845,000095:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
047846,000096: 
047847,000097:                                                                                                  #  ALARM OR ABORT EXIT MODES -
047848,000098:                                                                                                  #        TC      ABORT
047849,000099:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
047850,000100: 
047851,000101:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
047852,000102:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
047853,000103:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
047854,000104: 
047855,000105:                                                                                                  #  OUTPUT --
047856,000106:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
047857,000107: 
047858,000108:                                                                                                  #  DEBRIS -
047859,000109:                                                                                                  #        CENTRALS - A,Q,L
047860,000110:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
047861,000111: 
047862,000112:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
047863,000113:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
047864,000114:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
047865,000115:                                                                                                  #                LET TO = THE TIME OF THE TC WAITLIST
047866,000116:                                                                                                  #                LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
047867,000117:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
047868,000118:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
047869,000119:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
047870,000120:                                                                                                  #                         (Z=0, USUALLY)
047871,000121:                                                                                                  #                LET DELTD  = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
047872,000122:                                                                                                  #                THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
047873,000123:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
047874,000124:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
047875,000125:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
047876,000126:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
