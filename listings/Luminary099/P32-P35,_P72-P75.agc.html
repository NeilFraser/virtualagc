Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	16:27 JULY 14,1969
Note that the date is the date of the printout, not the date of the program revision.

026650,000045:                                                                                                  #  PURPOSE
026651,000046: 
026652,000047:                                                                                                  #        (1)     TO CALCULATE PARAMETERS ASSOCIATED WITH THE FOLLOWING
026653,000048:                                                                                                  #                CONCENTRIC FLIGHT PLAN MANEUVERS - THE CO-ELLIPTIC SEQUENCE
026654,000049:                                                                                                  #                INITIATION (CSI) MANEUVER AND THE CONSTANT DELTA ALTITUDE
026655,000050:                                                                                                  #                (CDH) MANEUVER.
026656,000051: 
026657,000052:                                                                                                  #        (2)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
026658,000053:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
026659,000054: 
026660,000055:                                                                                                  #        (3)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND DEPENDENT VARIABLES
026661,000056:                                                                                                  #                ASSOCIATED WITH THE CONCENTRIC FLIGNT PLAN MANEUVERS FOR
026662,000057:                                                                                                  #                APPROVAL BY THE ASTRONAUT/GROUND.
026663,000058: 
026664,000059:                                                                                                  #        (4)     TO STORE THE CSI TARGET PARAMETERS FOR USE BY THE DESIRED
026665,000060:                                                                                                  #                THRUSTING PROGRAM.
026666,000061: 
026667,000062:                                                                                                  #  ASSUMPTIONS
026668,000063: 
026669,000064:                                                                                                  #        (1)     AT A SELECTED TPI TIME THE LINE OF SIGHT BETWEEN THE ACTIVE
026670,000065:                                                                                                  #                AND PASSIVE VEHICLES IS SELECTED TO BE A PRESCRIBED ANGLE (E)
026671,000066:                                                                                                  #                FROM THE HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE
026672,000067:                                                                                                  #                POSITION.
026673,000068: 
026674,000069:                                                                                                  #        (2)     THE TIME BETWEEN CSI IGNITION AND CDH IGNITION MUST BE
026675,000070:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026676,000071:                                                                                                  #                COMPLETION OF THE PROGRAM.
026677,000072: 
026678,000073:                                                                                                  #        (3)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION MUST BE
026679,000074:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026680,000075:                                                                                                  #                COMPLETION OF THE PROGRAM.
026681,000076: 
026682,000077:                                                                                                  #        (4)     CDH DELTA V IS SELECTED TO MINIMIZE THE VARIATION OF THE
026683,000078:                                                                                                  #                ALTITUDE DIFFERENCE BETWEEN THE ORBITS.
026684,000079: 
026685,000080:                                                                                                  #        (5)     CSI BURN IS DEFINED SUCH THAT THE IMPULSIVE DELTA V IS IN THE
026686,000081:                                                                                                  #                HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION AT CSI
026687,000082:                                                                                                  #                IGNITION.
026688,000083: 
026689,000084:                                                                                                  #        (6)     THE PERICENTER ALTITUDE OF THE ORBIT FOLLOWING CSI AND CDH
026690,000085:                                                                                                  #                MUST BE GREATER THAN 35,000 FT (LUNAR ORBIT) OR 85 NM (EARTH
026691,000086:                                                                                                  #                ORBIT) FOR SUCCESSFUL COMPLETION OF THIS PROGRAM.
026692,000087: 
026693,000088:                                                                                                  #        (7)     THE CSI AND CDH MANEUVERS ARE ORIGINALLY ASSUMED TO BE
026694,000089:                                                                                                  #                PARALLEL TO THE PLANE OF THE CSM ORBIT.  HOWEVER CREW

026713,000108:                                                                                                  #                MARK COUNTER IS ZEROED BY THE SELECTION OF P20 AND AFTER EACH
026714,000109:                                                                                                  #                THRUSTING MANEUVER.

026719,000114:

026723,000118:                                                                                                  #                        EACH RENDEZVOUS PRE-THRUSTING PROGRAM.

026725,000120:                                                                                                  #                        FINAL FLAG - SELECTS FINAL PROGRAM DISPLAYS AFTER CREW HAS
026726,000121:                                                                                                  #                        COMPLETED THE FINAL MANEUVER COMPUTATION AND DISPLAY
026727,000122:                                                                                                  #                        CYCLE.

026731,000126:                                                                                                  #                        THRUSTING PROGRAM SELECTED AFTER COMPLETION OF THIS
026732,000127:                                                                                                  #                        PROGRAM.

026735,000130:                                                                                                  #                A THRUSTING MANEUVER.
