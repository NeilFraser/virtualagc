The printout was digitally photographed by Paul Fjeld. Note that the page images presented online are of reduced quality, and that higher-quality images are available. 055038,000032:                                                                                                  #  PROGRAM DESCRIPTION                           DATE   20 DECEMBER 1967
055039,000033:                                                                                                  #  PROGRAM NAME - SELF-CHECK                     LOG SECTION  AGC BLOCK TWO SELF-CHECK
055040,000034:                                                                                                  #  MOD NO - 1                                    ASSEMBLY SUBROUTINE UTILITYM REV 25
055041,000035:                                                                                                  #  MOD BY - GAUNTT
055042,000036: 
055043,000037:                                                                                                  #  FUNCTIONAL DESCRIPTION
055044,000038:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
055045,000039:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
055046,000040:                                                                                                  #        STARTING VERB.
055047,000041: 
055048,000042:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
055049,000043: 
055050,000044:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
055051,000045: 
055052,000046:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
055053,000047:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
055054,000048: 
055055,000049:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
055056,000050:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
055057,000051:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
055058,000052:                                                                                                  #                +-4             ERASABLE MEMORY
055059,000053:                                                                                                  #                +-5             FIXED MEMORY
055060,000054:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
055061,000055:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
055062,000056:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
055063,000057: 
055064,000058:                                                                                                  #  WARNINGS
055065,000059:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
055066,000060:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
055067,000061: 
055068,000062:                                                                                                  #  CALLING SEQUENCE
055069,000063:                                                                                                  #        TO CALL SELF-CHECK KEY IN
055070,000064:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
055071,000065:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
055072,000066:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
055073,000067:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
055074,000068: 
055075,000069:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
055076,000070:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
055077,000071:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
055078,000072: 
055079,000073:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
055080,000074: 
055081,000075:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
055082,000076: 
055083,000077:                                                                                                  #  OUTPUT

Page 1285 

055085,000079:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
055086,000080:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
055087,000081:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
055088,000082:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
055089,000083:                                                                                                  #        INITIATED FRESH START (SLAP1).
055090,000084: 
055091,000085:                                                                                                  #        SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
055092,000086:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
055093,000087: 
055094,000088:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
055095,000089:                                                                                                  #        ACCOMPLISHED BY FRESH START
055096,000090:                                                                                                  #                SMODE SET TO +0
055097,000091: 
055098,000092:                                                                                                  #  DEBRIS
055099,000093:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
055100,000094:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
055101,000095:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
