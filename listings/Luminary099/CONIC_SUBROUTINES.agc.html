049584,000045:                                                                                                  #  INVERSE-SQUARE FORCE ACTING ON A POINT MASS, AS OUTLINED IN THE CMC AND LGC LUNAR LANDING MISSION GSOP, SECTION
049585,000046:                                                                                                  #  5.5.1.2.  A GENERAL USAGE POINT-OF-VIEW WAS TAKEN IN FORMULATING, MECHANIZING, AND SCALING THE SUBROUTINES,
049586,000047:                                                                                                  #  RATHER THAN OPTIMIZING EACH FOR A PARTICULAR USE.  THEREFORE, MULTIPLE USAGE CAN BE MADE OF THE SUBROUTINES
049587,000048:                                                                                                  #  INVOLVING ANY REALISTIC SET OF CONSTRAINTS.  IT SHOULD BE NOTED THAT ONLY ONE SET OF CODING IS USED, WHETHER THE
049588,000049:                                                                                                  #  EARTH, MOON, OR ANY OTHER CELESTIAL BODY IS SPECIFIED AS THE CENTRAL BODY OF THE PROBLEM, PROVIDED ONE OBSERVES
049589,000050:                                                                                                  #  THE INHERENT SCALE CHANGE REQUIRED IN POSITION, VELOCITY, MU, AND TIME, AS OUTLINED IN MISSION PROGRAMMING
049590,000051:                                                                                                  #  DEFINITION MEMO NO. 10.  THIS CAN BE ACCOMPLISHED BY SIMPLY ADDING TO THE MUTABLE AND INITIALIZING THE SUBROU-
049591,000052:                                                                                                  #  TINES APPROPRIATELY.
049592,000053: 
049593,000054:                                                                                                  #        DUE TO THE UNIFORMITY OF THE EQUATIONS INVOLVED, CODING WAS MINIMIZED BY TREATING INDIVIDUAL EQUATIONS AND
049594,000055:                                                                                                  #  BLOCKS OF EQUATIONS AS SUBROUTINES OF LOWER RANK WHENEVER POSSIBLE.  AS A RESULT, THREE BY-PRODUCTS SUBROUTINES,
049595,000056:                                                                                                  #  DIRECTLY USABLE AS INDEPENDENT SUBROUTINES, WERE GENERATED.
049596,000057: 
049597,000058:                                                                                                  #  RESTRICTIONS -
049598,000059:                                                                                                  #        THE ONLY LIMITATION IN THE SCOPE OF PROBLEM WHICH CAN BE SOLVED BY A PARTICULAR SUBROUTINE IS THE SCALING
049599,000060:                                                                                                  #  LIMIT OF EACH PARAMETER AS SPECIFIED IN THE GSOP.  THESE SCALING LIMITS WERE CHOSEN SO THAT ALL FEASIBLE TRAJEC-
049600,000061:                                                                                                  #  TORIES COULD BE HANDLED.
