Source Code

These source-code files are part of a reconstructed copy of Comanche 72/3 (AKA Manche72 revision 3), the final, flown mission release of the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 13.

The starting point was the source code of Comanche 72 (itself previously reconstructed). Comanche 72 source code was then modified by incorporating changes known from contemporary documentation to have been made between revisions 72 and 72/3. The resulting code assembles to have identical memory-bank checksums as those specified in NASA drawing 2021153G; this gives reasonably high confidence that the changes that have been made are correct. In places where ambiguity exists, this ambiguity is discussed in "## Reconstruction:" comments.

Since no contemporary listings of Comanche 72/3 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

008586,000019:                                                                                                  #  RESTART TABLES
008587,000020:                                                                                                  #  ------------------
008588,000021: 
008589,000022:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
008590,000023:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
008591,000024:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
008592,000025:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
008593,000026: 
008594,000027:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN PRDTTAB OF THE CORRECT PHASE SPOT - A POSITIVE PRIORITY INDICATES A
008595,000028:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
008596,000029:                                                                                                  #  FOR EXAMPLE,
008597,000030: 
008598,000031:                                                                                                  #                5.7SPOT         OCT     23000
008599,000032:                                                                                                  #                                2CADR   SOMEJOB
008600,000033: 
008601,000034:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
008602,000035: 
008603,000036:                                                                                                  #                5.5SPOT         OCT     -23000
008604,000037:                                                                                                  #                                2CADR   ANYJOB
008605,000038: 
008606,000039:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
008607,000040:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
008608,000041:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
008609,000042:                                                                                                  #  WITH.  EXAMPLE,
008610,000043: 
008611,000044:                                                                                                  #                3.6SPOT         GENADR  DELTAT
008612,000045:                                                                                                  #                                -GENADR LONGTASK
008613,000046:                                                                                                  #                                BBCON   LONGTASK
008614,000047: 
008615,000048:                                                                                                  #                                OCT     31000
008616,000049:                                                                                                  #                                2CADR   JOBAGAIN
008617,000050: 
008618,000051:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
008619,000052:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATOIN SHOULD BE IN THE BBCON OF THE 2CADR OF THE
008620,000053:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
008621,000054: 
008622,000055:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
008623,000056:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
008624,000057:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY. IT SHOULD BE NOTED AS ABOVE, THAT
008625,000058:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
008626,000059:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
008627,000060:                                                                                                  #  TASK.  EXAMPLES,
008628,000061: 
008629,000062:                                                                                                  #                                OCT     77777           THIS WILL CAUSE AN IMMEDIATE RESTART
008630,000063:                                                                                                  #                                -2CADR  ATASK           OF THE TASK :ATASK:
008631,000064: 
008632,000065:                                                                                                  #                                DEC     200             IF THE TIME OF THE 2 SECONDS SINCE DUMMY
008633,000066:                                                                                                  #                                -2CADR  DUMMY           WAS PUT ON WAITLIST IS UP, IT WILL BEGIN
008634,000067:                                                                                                  #                                                        IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
008635,000068:                                                                                                  #                                                        IT NORMALLY WOULD HAVE BEGUN.
008636,000069:                                                                                                  #                                -GENADR DTIME           WHERE DTIME CONTAINS THE DELTA TIME
008637,000070:                                                                                                  #                                -2CADR  TASKTASK        OTHERWISE THIS IS AS ABOVE
