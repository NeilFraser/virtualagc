Source Code

These source-code files are part of a reconstructed copy of Comanche 67, the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 12.

They have been adapted from sources files for Comanche 55, and updated via disassembly of binary dumps of original core rope memory modules, part numbers 2003972-1081, 2003972-1091, 2003972-1111, 2003972-1121, 2003972-1131, and 2003972-1141.

Since only binary dumps (rather than listings) of Comanche 67 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

006733,000038:                                                                                                  #  DOWNLIST FORMAT DEFINITIONS AND RULES-
006734,000039:                                                                                                  #  1. END OF A LIST = -XDNADR (X = 1 TO 6), -DNPTR, OR -DNCHAN.
006735,000040:                                                                                                  #  2. SNAPSHOT SUBLIST = LIST WHICH STARTS WITH A -1DNADR.
006736,000041:                                                                                                  #  3. SNAPSHOT SUBLIST CAN ONLY CONTAIN 1DNADRS.
006737,000042:                                                                                                  #  4. TIME2 1DNADR MUST BE LOCATED IN THE CONTROL LIST OF A DOWNLIST.
006738,000043:                                                                                                  #  5. ERASABLE DOWN TELEMETRY WORDS SHOULD BE GROUPED IN SEQUENTIAL
006739,000044:                                                                                                  #     LOCATIONS AS MUCH AS POSSIBLE TO SAVE STORAGE USED BY DOWNLINK LISTS.
006740,000045:                                                                                                  #  6. THE DOWNLINK LISTS (INCLUDING SUBLISTS) ARE ORGANIZED SUCH THAT THE ITEMS LISTED FIRST (IN FRONT OF FBANK) ARE
006741,000046:                                                                                                  #     SENT FIRST.  EXCEPTION--- SNAPSHOT SUBLISTS.  IN THE SNAPSHOT SUBLISTS THE DATA REPRESENTED BY THE FIRST
006742,000047:                                                                                                  #     11 1DNADRS IS PRESERVED (IN ORDER) IN DNTMBUFF AND SENT BY THE NEXT 11 DOWNRUPTS.  THE DATA REPRESENTED BY THE
006743,000048:                                                                                                  #     LIST IS SENT IMMEDIATELY.
