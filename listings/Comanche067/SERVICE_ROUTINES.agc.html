Source Code

These source-code files are part of a reconstructed copy of Comanche 67, the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 12.

They have been adapted from sources files for Comanche 55, and updated via disassembly of binary dumps of original core rope memory modules, part numbers 2003972-1081, 2003972-1091, 2003972-1111, 2003972-1121, 2003972-1131, and 2003972-1141.

Since only binary dumps (rather than listings) of Comanche 67 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

061559,000060:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
061560,000061:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
061561,000062:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
061562,000063:                                                                                                  #  ASSEMBLER.
061563,000064: 
061564,000065:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
061565,000066:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
061566,000067: 
061567,000068:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS:-
061568,000069:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
061569,000070:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
061570,000071:

061671,000172:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
061672,000173:                                                                                                  #  WRITTEN BY D. EYLES
061673,000174:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
061674,000175:                                                                                                  #  MOD 2 BY SCHULENBERG  (REMOVE RELINT)  SKIPPER REV 4 2/28/68
061675,000176: 
061676,000177:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
061677,000178:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
061678,000179: 
061679,000180:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
061680,000181:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
061681,000182:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
061682,000183:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
061683,000184:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
061684,000185:                                                                                                  #        I +4                            RETURNS HERE.
061685,000186: 
061686,000187:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
061687,000188:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
