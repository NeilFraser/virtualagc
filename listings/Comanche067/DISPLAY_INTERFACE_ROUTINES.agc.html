Source Code

These source-code files are part of a reconstructed copy of Comanche 67, the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 12.

They have been adapted from sources files for Comanche 55, and updated via disassembly of binary dumps of original core rope memory modules, part numbers 2003972-1081, 2003972-1091, 2003972-1111, 2003972-1121, 2003972-1131, and 2003972-1141.

Since only binary dumps (rather than listings) of Comanche 67 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

060081,000022:                                                                                                  #            SEQUENCE OF EVENTS.
060082,000023:                                                                                                  #        4.  MISC. DISPLAYS - ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES.  THESE INCLUDE SUCH DISPLAYS AS
060083,000024:                                                                                                  #            MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
060084,000025:                                                                                                  #        5.  ASTRONAUT INITIATED DISPLAYS - ALL DISPLAYS INITIATED EXTERNALLY.
060085,000026: 
060086,000027:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
060087,000028:                                                                                                  #        1.  ACTIVE - THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
060088,000029:                                                                                                  #            (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
060089,000030:                                                                                                  #            BUT NOT WAITING FOR A RESPONSE.
060090,000031:                                                                                                  #        2.  INACTIVE - A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
060091,000032:                                                                                                  #            (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
060092,000033:                                                                                                  #            WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
060093,000034:                                                                                                  #            OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
060094,000035:                                                                                                  #            HAS INFO SAVED FOR RESTART PURPOSES.
060095,000036: 
060096,000037:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS-
060097,000038:                                                                                                  #        INTERRUPTS -
060098,000039:                                                                                                  #                1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
060099,000040:                                                                                                  #                2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
060100,000041:                                                                                                  #                3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
060101,000042:                                                                                                  #                    PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
060102,000043:                                                                                                  #                4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
060103,000044:                                                                                                  #                5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
060104,000045: 
060105,000046:                                                                                                  #        ORDER OF WAITING DISPLAYS-
060106,000047:                                                                                                  #                1.  ASTRONAUT EXTERNAL USE
060107,000048:                                                                                                  #                2.  PRIORITY
060108,000049:                                                                                                  #                3.  INTERRUPTED MARK
060109,000050:                                                                                                  #                4.  INTERRUPTED NORMAL
060110,000051:                                                                                                  #                5.  MARK TO BE REQUESTED (SEE DESCRIPTION OF ENDMARK)
060111,000052:                                                                                                  #                6.  MARK WAITING
060112,000053:                                                                                                  #                7.  NORMAL WAITING

060114,000055:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
060115,000056:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
060116,000057:                                                                                                  #  WILL FOLLOW LATER):
060117,000058:                                                                                                  #        1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
060118,000059:                                                                                                  #            IS TO THE USERS CALL CADR +4.  FOR THE ONLY NON FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
060119,000060:                                                                                                  #            IS TO THE USERS CALLING LOC +1.

060121,000062:                                                                                                  #        3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USERS JOB PRIORITY.
060122,000063:                                                                                                  #        4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
060123,000064:                                                                                                  #            USERS JOB.
060124,000065:                                                                                                  #        5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
060125,000066:                                                                                                  #        6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
060126,000067:                                                                                                  #            DESCRIPTION OF RESTARTS).

060128,000069:                                                                                                  #            TO THE USERS CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USERS CALL CADR +2.  AN ENTER OR RECYCLE
060129,000070:                                                                                                  #            (V32) BRANCHES TO THE USERS CALL CADR +3.
