Source Code

These source-code files are part of a reconstructed copy of Comanche 67, the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 12.

They have been adapted from sources files for Comanche 55, and updated via disassembly of binary dumps of original core rope memory modules, part numbers 2003972-1081, 2003972-1091, 2003972-1111, 2003972-1121, 2003972-1131, and 2003972-1141.

Since only binary dumps (rather than listings) of Comanche 67 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

046311,000029:                                                                                                  #  FUNCTIONAL DESCRIPTION - THIS ROUTINE IS INITIATED BY TELEMETRY END
046312,000030:                                                                                                  #        PULSE FROM THE DOWNLINK TELEMETRY CONVERTER.  THIS PULSE OCCURS
046313,000031:                                                                                                  #        AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
046314,000032:                                                                                                  #        EXECUTED AT THESE RATES.  THIS ROUTINE SELECTS THE APPROPRIATE
046315,000033:                                                                                                  #        AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
046316,000034:                                                                                                  #        CHANNELS 34 AND 35.  THE INFORMATION IS THEN GATED OUT FROM THE
046317,000035:                                                                                                  #        LGC IN SERIAL FASHION.
046318,000036: 
046319,000037:                                                                                                  #        THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST.  SINCE DOWNRUPTS
046320,000038:                                                                                                  #        OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
046321,000039:                                                                                                  #        CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
046322,000040:                                                                                                  #        OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
046323,000041: 
046324,000042:                                                                                                  #  CALLING SEQUENCE - NONE
046325,000043:                                                                                                  #        PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
046326,000044:                                                                                                  #        RESULT OF A DOWNRUPT.  CONTROL IS RETURNED VIA TCF RESUME WHICH
046327,000045:                                                                                                  #        IN EFFECT IS A RESUME.
046328,000046: 
046329,000047:                                                                                                  #  SUBROUTINES CALLED - NONE
046330,000048: 
046331,000049:                                                                                                  #  NORMAL EXIT MODE - TCF RESUME
046332,000050: 
046333,000051:                                                                                                  #  ALARM OR ABORT EXIT MODE - NONE
046334,000052: 
046335,000053:                                                                                                  #  RESTART PROTECTION:
046336,000054:                                                                                                  #        ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
046337,000055:                                                                                                  #        DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E. CURRENT CONTENTS OF DNLSTADR).  THIS HAS THE
046338,000056:                                                                                                  #        EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
046339,000057:                                                                                                  #        THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
046340,000058:                                                                                                  #        DOWNLIST.
046341,000059: 
046342,000060:                                                                                                  #        ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
046343,000061: 
046344,000062:                                                                                                  #  DOWNLINK LIST SELECTION:
046345,000063:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
046346,000064:                                                                                                  #        1.      FRESH START
046347,000065:                                                                                                  #        2.      V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
046348,000066:                                                                                                  #        3.      UPDATE PROGRAM (P27)
046349,000067:                                                                                                  #        4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
046350,000068:                                                                                                  #                TRANSITION (COLOSSUS) ETC.).
046351,000069: 
046352,000070:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
046353,000071:                                                                                                  #        READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
046354,000072: 
046355,000073:                                                                                                  #  OUTPUT - EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
046356,000074:                                                                                                  #        COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
046357,000075: 
046358,000076:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED - NONE
046359,000077:                                                                                                  #        'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
046360,000078: 
046361,000079:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM) -
046362,000080:                                                                                                  #        LDATALST, DNTMBUFF TO DNTMBUFF +21D, TMINDEX, DNQ.

046365,000083:                                                                                                  #  RECEIPT OF AN ENDPULSE FROM THE SPACECRAFT TELEMETRY PROGRAMMER.
046366,000084: 
046367,000085:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
046368,000086:                                                                                                  #  1.    DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
046369,000087:                                                                                                  #        DOWN-TELEMETRY PROGRAM.  THIS IS DONE FOR EASE OF CODING, FASTER
046370,000088:                                                                                                  #        EXECUTION.
046371,000089:                                                                                                  #  2.    EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
046372,000090:                                                                                                  #        SUBLISTS.
046373,000091:                                                                                                  #  3.    A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
046374,000092:                                                                                                  #        DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
046375,000093:                                                                                                  #        CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
046376,000094:                                                                                                  #  4.    SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
046377,000095:                                                                                                  #        SAVED IN A BUFFER DURING ONE DOWNRUPT.
046378,000096:                                                                                                  #  5.    THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
046379,000097:                                                                                                  #        OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
046380,000098:                                                                                                  #        IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
046381,000099:                                                                                                  #  6.    ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
046382,000100:                                                                                                  #        CREATED FOR THE DOWNLIST PROGRAM.
046383,000101:                                                                                                  #        A.      1DNADR                  1-WORD DOWNLIST ADDRESS.
046384,000102:                                                                                                  #                SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
046385,000103:                                                                                                  #                HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
046386,000104:                                                                                                  #        B.      2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
046387,000105:                                                                                                  #                SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
046388,000106:                                                                                                  #                FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
046389,000107:                                                                                                  #                PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
046390,000108:                                                                                                  #        C.      DNCHAN                  DOWNLIST CHANNEL ADDRESS.
046391,000109:                                                                                                  #                SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
046392,000110:                                                                                                  #                A PAIR OF CHANNELS FOR DOWN TELEMETRY.
046393,000111:                                                                                                  #        D.      DNPTR                   DOWN TELEMETRY SUBLIST POINTER.
046394,000112:                                                                                                  #                SAME AS CAF BUT TAGGED AS A CONSTANT.  USED IN CONTROL LIST TO POINT TO A SUBLIST.
046395,000113:                                                                                                  #                CAUTION --- A DNPTR CANNOT BE USED IN A SUBLIST.
046396,000114:                                                                                                  #  7.    THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
046397,000115:                                                                                                  #        A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
046398,000116:                                                                                                  #  8.    IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

046411,000129:                                                                                                  #        CONTROL SECTION OF THE DOWNLIST.
046412,000130:                                                                                                  #  5.    'DNCHAN 0' CANNOT BE USED.
046413,000131:                                                                                                  #  6.    'DNPTR 0' CANNOT BE USED.
046414,000132:                                                                                                  #  7.    DNPTR CANNOT APPEAR IN A SUBLIST.
