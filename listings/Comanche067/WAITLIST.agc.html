Source Code

These source-code files are part of a reconstructed copy of Comanche 67, the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 12.

They have been adapted from sources files for Comanche 55, and updated via disassembly of binary dumps of original core rope memory modules, part numbers 2003972-1081, 2003972-1091, 2003972-1111, 2003972-1121, 2003972-1131, and 2003972-1141.

Since only binary dumps (rather than listings) of Comanche 67 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

050689,000024:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),

050693,000028:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050694,000029: 
050695,000030:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
050696,000031:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
050697,000032:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
050698,000033:                                                                                                  #                                        .
050699,000034:                                                                                                  #                                        .
050700,000035:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
050701,000036:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
050702,000037: 
050703,000038:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
050704,000039:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
050705,000040:                                                                                                  #                                        .
050706,000041:                                                                                                  #                                        .
050707,000042:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
050708,000043:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9

050711,000046:                                                                                                  #  --------
050712,000047:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
050713,000048:                                                                                                  #        2)      9 TASKS MAXIMUM
050714,000049:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
050715,000050:                                                                                                  #        4)      TASKS END BY TC TASKOVER

050718,000053:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
050719,000054:                                                                                                  #        L       TC      WAITLIST
050720,000055:                                                                                                  #        L+1     2CADR   DESIRED TASK

050724,000059:                                                                                                  #  TWIDDLE-
050725,000060:                                                                                                  #  -------
050726,000061:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
050727,000062:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,
050728,000063:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
050729,000064:                                                                                                  #                L-1     CA      DELTAT
050730,000065:                                                                                                  #                L       TC      TWIDDLE
050731,000066:                                                                                                  #                L+1     ADRES   DESIRED TASK
050732,000067:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
050733,000068: 
050734,000069:                                                                                                  #  NORMAL EXIT MODES -
050735,000070:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
050736,000071: 
050737,000072:                                                                                                  #  ALARM OR ABORT EXIT MODES -
050738,000073:                                                                                                  #        TC      ABORT
050739,000074:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
050740,000075: 
050741,000076:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
050742,000077:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
050743,000078:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
050744,000079:

050746,000081:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
050747,000082: 
050748,000083:                                                                                                  #  DEBRIS -
050749,000084:                                                                                                  #        CENTRALS - A,Q,L
050750,000085:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050751,000086: 
050752,000087:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
050753,000088:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
050754,000089:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
050755,000090:                                                                                                  #                LET TO = THE TIME OF THE TC WAITLIST
050756,000091:                                                                                                  #                LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
050757,000092:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050758,000093:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050759,000094:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050760,000095:                                                                                                  #                (Z=0, USUALLY)
050761,000096:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050762,000097:                                                                                                  #        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050763,000098:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050764,000099:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050765,000100:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050766,000101:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
