Source Code

These source-code files are part of a reconstructed copy of Comanche 67, the Apollo Guidance Computer (AGC) Command Module (CM) software for Apollo 12.

They have been adapted from sources files for Comanche 55, and updated via disassembly of binary dumps of original core rope memory modules, part numbers 2003972-1081, 2003972-1091, 2003972-1111, 2003972-1121, 2003972-1131, and 2003972-1141.

Since only binary dumps (rather than listings) of Comanche 67 are available as source material, all comments and labels are approximate. They have been taken from other AGC programs where possible, or, in some places, written from scratch to match what we believe would have been in the original listing.

008533,000017:                                                                                                  #  RESTART TABLES
008534,000018:                                                                                                  #  ------------------
008535,000019: 
008536,000020:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
008537,000021:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
008538,000022:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
008539,000023:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
008540,000024: 
008541,000025:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN PRDTTAB OF THE CORRECT PHASE SPOT - A POSITIVE PRIORITY INDICATES A
008542,000026:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
008543,000027:                                                                                                  #  FOR EXAMPLE,
008544,000028: 
008545,000029:                                                                                                  #                5.7SPOT         OCT     23000
008546,000030:                                                                                                  #                                2CADR   SOMEJOB
008547,000031: 
008548,000032:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
008549,000033: 
008550,000034:                                                                                                  #                5.5SPOT         OCT     -23000
008551,000035:                                                                                                  #                                2CADR   ANYJOB
008552,000036: 
008553,000037:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
008554,000038:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
008555,000039:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
008556,000040:                                                                                                  #  WITH.  EXAMPLE,
008557,000041: 
008558,000042:                                                                                                  #                3.6SPOT         GENADR  DELTAT
008559,000043:                                                                                                  #                                -GENADR LONGTASK
008560,000044:                                                                                                  #                                BBCON   LONGTASK
008561,000045: 
008562,000046:                                                                                                  #                                OCT     31000
008563,000047:                                                                                                  #                                2CADR   JOBAGAIN
008564,000048: 
008565,000049:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
008566,000050:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATOIN SHOULD BE IN THE BBCON OF THE 2CADR OF THE
008567,000051:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
008568,000052: 
008569,000053:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
008570,000054:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
008571,000055:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY. IT SHOULD BE NOTED AS ABOVE, THAT
008572,000056:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
008573,000057:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
008574,000058:                                                                                                  #  TASK.  EXAMPLES,
008575,000059: 
008576,000060:                                                                                                  #                                OCT     77777           THIS WILL CAUSE AN IMMEDIATE RESTART
008577,000061:                                                                                                  #                                -2CADR  ATASK           OF THE TASK :ATASK:
008578,000062: 
008579,000063:                                                                                                  #                                DEC     200             IF THE TIME OF THE 2 SECONDS SINCE DUMMY
008580,000064:                                                                                                  #                                -2CADR  DUMMY           WAS PUT ON WAITLIST IS UP, IT WILL BEGIN
008581,000065:                                                                                                  #                                                        IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
008582,000066:                                                                                                  #                                                        IT NORMALLY WOULD HAVE BEGUN.
008583,000067:                                                                                                  #                                -GENADR DTIME           WHERE DTIME CONTAINS THE DELTA TIME
008584,000068:                                                                                                  #                                -2CADR  TASKTASK        OTHERWISE THIS IS AS ABOVE
