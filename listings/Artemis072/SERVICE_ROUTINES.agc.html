Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

063562,000039:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
063563,000040:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
063564,000041:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
063565,000042:                                                                                                  #  ASSEMBLER.
063566,000043: 
063567,000044:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
063568,000045:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
063569,000046: 
063570,000047:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS:-
063571,000048:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
063572,000049:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
063573,000050: 
063574,000051:                                                                                                  #  RETURN IS TO THE LOCATION FOLLOWING THE "ADRES" ABOUT .58 MS AFTER THE "TC".
063575,000052:                                                                                                  #  UPON RETURN A CONTAINS THE CURRENT FLAGWRD SETTING.

063677,000154:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
063678,000155: 
063679,000156:                                                                                                  #  WRITTEN BY D. EYLES
063680,000157:                                                                                                  #  MOD 1 BY KERNAN                                               UTILITYM REV 17 11/18/67
063681,000158: 
063682,000159:                                                                                                  #  MOD 2 BY SCHULENBERG  (REMOVE RELINT)   SKIPPER REV 4 2/28/68
063683,000160: 
063684,000161:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
063685,000162:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
063686,000163: 
063687,000164:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
063688,000165: 
063689,000166:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
063690,000167:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
063691,000168:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
063692,000169:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
063693,000170:                                                                                                  #        I +4                            RETURNS HERE.
063694,000171: 
063695,000172:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
063696,000173: 
063697,000174:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
063698,000175:

063700,000177:    4000                                           BANK                                           
063701,000178: 
063702,000179:    5607     0061                                  EBANK=   ITEMP1                                
063703,000180: 
063704,000181:    5607                                           COUNT*   $$/TRAN                               
063705,000182: 
063706,000183:    5607           00004        GENTRAN            INHINT                                         
063707,000184:    5610           54061                 +1        TS       ITEMP1                                #  SAVE N-1
063708,000185:    5611           50002                           INDEX    Q                                     #  C(Q) = ADRES L.
063709,000186:    5612           60000                           AD       0                                     #  ADRES (L + N - 1).
