Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

014018,000050:                                                                                                  #  FUNCTIONAL DESCRIPTION-
014019,000051: 
014020,000052:                                                                                                  #        THE KEYBOARD AND DISPLAY SYSTEM PROGRAM OPERATES UNDER EXECUTIVE
014021,000053:                                                                                                  #        CONTROL AND PROCESSES INFORMATION EXCHANGED BETWEEN THE AGC AND THE
014022,000054:                                                                                                  #        COMPUTER OPERATOR.  THE INPUTS TO THE PROGRAM ARE FROM THE KEYBOARD,
014023,000055:                                                                                                  #        FROM INTERNAL PROGRAMS, AND FROM THE UPLINK.
014024,000056: 
014025,000057:                                                                                                  #        THE LANGUAGE OF COMMUNICATION WITH THE PROGRAM IS A PAIR OF WORDS
014026,000058:                                                                                                  #        KNOWN AS VERB AND NOUN.  EACH OF THESE IS REPRESENTED BY A 2 CHARACTER
014027,000059:                                                                                                  #        DECIMAL NUMBER.  THE VERB CODE INDICATES WHAT ACTION IS TO BE TAKEN, THE
014028,000060:                                                                                                  #        NOUN CODE INDICATES TO WHAT THIS ACTION IS APPLIED.  NOUNS USUALLY
014029,000061:                                                                                                  #        REFER TO A GROUP OF ERASABLE REGISTERS.
014030,000062: 
014031,000063:                                                                                                  #        VERBS ARE GROUPED INTO DISPLAYS, LOADS, MONITORS (DISPLAYS THAT ARE
014032,000064:                                                                                                  #        UPDATED ONCE PER SECOND), SPECIAL FUNCTIONS, AND EXTENDED VERBS (THESE
014033,000065:                                                                                                  #        ARE OUTSIDE OF THE DOMAIN OF PINBALL AND CAN BE FOUND UNDER LOG SECTION
014034,000066:                                                                                                  #        :EXTENDED VERBS:).
014035,000067: 
014036,000068:                                                                                                  #        A LIST OF VERBS AND NOUNS IS GIVEN IN LOG SECTION :ASSEMBLY AND
014037,000069:                                                                                                  #        OPERATION INFORMATION:.
014038,000070: 
014039,000071:                                                                                                  #  CALLING SEQUENCES-
014040,000072: 
014041,000073:                                                                                                  #        KEYBOARD:
014042,000074:                                                                                                  #        EACH DEPRESSION OF A MAIN (NAVIGATION) KEYBOARD BUTTON ACTIVATES
014043,000075:                                                                                                  #        INTERRUPT KEYRUPT1 (KEYRUPT2) AND PLACES THE 5 BIT KEY CODE INTO
014044,000076:                                                                                                  #        CHANNEL 15 (CHANNEL 16).  KEYRUPT1 (KEYRUPT2) PLACES THE KEY
014045,000077:                                                                                                  #        CODE INTO MPAC, ENTERS AN EXECUTIVE REQUEST FOR THE KEYBOARD AND DISPLAY
014046,000078:                                                                                                  #        PROGRAM (AT :CHARIN:), AND EXECUTES A RESUME.
014047,000079: 
014048,000080:                                                                                                  #        UPLINK:
014049,000081:                                                                                                  #        EACH WORD RECEIVED BY THE UPLINK ACTIVATES INTERRUPT UPRUPT WHICH
014050,000082:                                                                                                  #        PLACES THE 5 BIT KEY CODE INTO MPAC, ENTERS AN EXECUTIVE REQUEST FOR THE
014051,000083:                                                                                                  #        KEYBOARD AND DISPLAY PROGRAM (AT :CHARIN:) AND EXECUTES A RESUME.
014052,000084: 
014053,000085:                                                                                                  #        INTERNAL PROGRAMS:
014054,000086:                                                                                                  #        INTERNAL PROGRAMS CALL PINBALL AT :NVSUB: WITH THE DESIRED VERB/NOUN
014055,000087:                                                                                                  #        CODE IN A (LOW 7 BITS FOR NOUN, NEXT 7 BITS FOR VERB).  DETAILS
014056,000088:                                                                                                  #        DESCRIBED ON REMARKS CARDS JUST BEFORE :NVSUB: AND :NVSBWAIT: (SEE
014057,000089:                                                                                                  #        SYMBOL TABLE FOR PAGE NUMBERS).
014058,000090: 
014059,000091:                                                                                                  #  NORMAL EXIT MODES-
014060,000092: 
014061,000093:                                                                                                  #        IF PINBALL WAS CALLED BY EXTERNAL ACTION, THERE ARE FOUR EXITS:
014062,000094:                                                                                                  #                1) ALL BUT (2), (3), AND (4) EXIT DIRECTLY TO ENDOFJOB.
014063,000095:                                                                                                  #                2) EXTENDED VERBS GO TO THE EXTENDED VERB FAN AS PART OF THE

014066,000098:                                                                                                  #                   RESPONSIBILITY OF THE EXTENDED VERB CALLED TO EVENTUALLY
014067,000099:                                                                                                  #                   CHANGE PRIORITY (IF NECESSARY) AND DO AN ENDOFJOB.
014068,000100:                                                                                                  #                   ALSO PINBALL IS A NOVAC JOB. EBANK SET FOR COMMON.
