Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

041576,000019:                                                                                                  #  NAME          TVCDAPON (TVC DAP INITIALIZATION AND STARTUP CALL)
041577,000020:                                                                                                  #  LOG SECTION...TVCINITIALIZE                   SUBROUTINE...DAPCSM
041578,000021:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
041579,000022:                                                                                                  #  MODIFIED BY BEALS TO ELIMINATE CSMMASS UPDATE LOGIC (NOW DONE IN S40.8)
041580,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION
041581,000024:                                                                                                  #        PERFORMS TVCDAP INITIALIZATION (GAINS, TIMING PARAMETERS, FILTER VARIABLES, ETC.)

041583,000026:                                                                                                  #        MAKES INITIALIZATION CALL TO ..NEEDLER.. FOR TVC DAP NEEDLES-SETUP

041585,000028:                                                                                                  #        CALLS TVCEXECUTIVE AT TVCEXEC, VIA WAITLIST
041586,000029:                                                                                                  #        CALLS TVCDAP CDU-RATE INITIALIZATION PKG AT DAPINIT  VIA T5
041587,000030:                                                                                                  #        PROVIDES FOR LOADING OF LOW-BANDWIDTH COEFFS AND GAINS AT SWICHOVR
041588,000031:                                                                                                  #  CALLING SEQUENCE - T5LOC=2CADR(TVCDAPON,EBANK=BZERO), T5=.6SECT5
041589,000032:                                                                                                  #        IN PARTICULAR, CALLED BY ..DOTVCON.. IN P40
041590,000033:                                                                                                  #        MRCLEAN AND TVCINIT4 ARE POSSIBLE RESTART ENTRY POINTS
041591,000034:                                                                                                  #  NORMAL EXIT MODE
041592,000035:                                                                                                  #        TCF RESUME
041593,000036:                                                                                                  #  SUBROUTINES CALLED
041594,000037:                                                                                                  #        NEEDLER, MASSPROP
041595,000038:                                                                                                  #  ALARM OR ABORT EXIT MODES
041596,000039:                                                                                                  #        NONE
041597,000040:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
041598,000041:                                                                                                  #        CSMMASS, LEMMASS, DAPDATR1 (FOR MASSPROP SUBROUTINE)
041599,000042:                                                                                                  #        TVC PAD LOADS (SEE EBANK6 IN ERASABLE ASSIGNMENTS)
041600,000043:                                                                                                  #        PACTOFF, YACTOFF, CDUX
041601,000044:                                                                                                  #        TVCPHASE AND THE T5 BITS OF FLAGWRD6 (SET AT DOTVCON IN P40)
041602,000045:                                                                                                  #  OUTPUT
041603,000046:                                                                                                  #        ALL TVC AND ROLL DAP ERASABLES, FLAGWRD6 (BITS 13,14), T5, WAITLIST
