Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

022743,000028:                                                                                                  #     USED THROUGHOUT.              CALCULATE AND DISPLAY ORBITAL PARAMETERS
022744,000029: 
022745,000030:                                                                                                  #  1. IF AVERAGE G IS OFF:
022746,000031:                                                                                                  #          FLASH DISPLAY V04N12. R2 INDICATES WHICH SHIP'S STATE VECTOR IS
022747,000032:                                                                                                  #           TO BE UPDATED. INITIAL CHOICE IS THIS SHIP (R2=1). ASTRONAUT
022748,000033:                                                                                                  #           CAN CHANGE TO OTHER SHIP BY V22EXE, WHERE X NOT EQ 1.
022749,000034:                                                                                                  #           FLASH DISPLAY TIME V06N16. INITIAL CHOICE = 0 = PRESENT TIME.
022750,000035:                                                                                                  #           ASTRONAUT CAN INSERT A TIME BY V25E, R1=HR R2=MIN R3=CSEC.
022751,000036:                                                                                                  #           CHOSEN STATE VECTOR UPDATED TO CHOSEN TIME BY THISPREC/OTHPREC
022752,000037:                                                                                                  #          CALLS SR30.1 (WHICH CALLS TFFCONMU + TFFRP/RA) TO CALCULATE
022753,000038:                                                                                                  #           RPER (PERIGEE RADIUS), RAPO (APOGEE RADIUS), HPER (PERIGEE
022754,000039:                                                                                                  #           HEIGHT ABOVE LAUNCH PAD OR LUNAR LANDING SITE), HAPO (APOGEE
022755,000040:                                                                                                  #           HEIGHT AS ABOVE), TPER (TIME TO PERIGEE), TFF (TIME TO
022756,000041:                                                                                                  #           INTERSECT 300 KFT ABOVE PAD OR 35KFT ABOVE LANDING SITE).
022757,000042:                                                                                                  #          FLASH MONITOR V16N44 (HAPO, HPER, TFF).TFF IS -59M59S IF IT WAS
022758,000043:                                                                                                  #           NOT COMPUTABLE, OTHERWISE IT INCREMENTS ONCE PER SECOND.
022759,000044:                                                                                                  #           IF P00 IS RUNNING, CALL DELRSPL SPLASH ERROR COMPUTATION.
022760,000045:                                                                                                  #           ASTRONAUT HAS OPTION TO MONITOR TPER BY KEYING IN N 32 E.
022761,000046:                                                                                                  #           DISPLAY IS IN HMS, IS NEGATIVE (AS WAS TFF), AND INCREMENTS
022762,000047:                                                                                                  #           ONCE PER SECOND ONLY IF TFF DISPLAY WAS -59M59S.
022763,000048: 
022764,000049:                                                                                                  #  2. IF AVERAGE G IS ON:
022765,000050:                                                                                                  #          CALLS SR30.1 APPROX EVERY TWO SECS.  STATE VECTOR IS ALWAYS
022766,000051:                                                                                                  #           FOR THIS VEHICLE. V82 DOES NOT DISTURB STATE VECTOR.  RESULTS
022767,000052:                                                                                                  #           OF SR30.1 ARE RAPO, RPER, HAPO, HPER, TPER, TFF.
022768,000053:                                                                                                  #          FLASH MONITOR V16N44 (HAPO, HPER, TFF).
022769,000054:                                                                                                  #           IF MODE IS P11, THEN CALL DELRSPL SO ASTRONAUT CAN MONITOR
022770,000055:                                                                                                  #           RESULTS BY N50E. SPLASH COMPUTATION DONE ONCE PER TWO SECS.
022771,000056:
