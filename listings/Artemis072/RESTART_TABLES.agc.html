Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

010067,000024: 
010068,000025:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
010069,000026:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
010070,000027:                                                                                                  #  EACH USING THREE LOCATIONS. THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
010071,000028:                                                                                                  #  WAY THINGS ARE PUT INTO THE TABLES.
010072,000029:                                                                                                  #       A JOB HAS ITS PRIORITY STORED IN PRDTTAB OF THE CORRECT PHASE SPOT - A POSITIVE PRIORITY INDICATES A
010073,000030:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
010074,000031:                                                                                                  #  FOR EXAMPLE,
010075,000032: 
010076,000033:                                                                                                  #                5.7SPOT         OCT     23000
010077,000034:                                                                                                  #                                2CADR   SOMEJOB
010078,000035: 
010079,000036:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
010080,000037: 
010081,000038:                                                                                                  #                5.5SPOT         OCT     -23000
010082,000039:                                                                                                  #                                2CADR   ANYJOB
010083,000040: 
010084,000041:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
010085,000042:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
010086,000043:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
010087,000044:                                                                                                  #  WITH.  EXAMPLE,
010088,000045: 
010089,000046:                                                                                                  #                3.6SPOT         GENADR  DELTAT
010090,000047:                                                                                                  #                                -GENADR LONGTASK
010091,000048:                                                                                                  #                                BBCON   LONGTASK
010092,000049: 
010093,000050:                                                                                                  #                                OCT     31000
010094,000051:                                                                                                  #                                2CADR   JOBAGAIN
010095,000052: 
010096,000053:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED. IT SHOULD
010097,000054:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATOIN SHOULD BE IN THE BBCON OF THE 2CADR OF THE
010098,000055:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
010099,000056: 
010100,000057:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
010101,000058:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
010102,000059:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY. IT SHOULD BE NOTED AS ABOVE, THAT
010103,000060:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
010104,000061:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
010105,000062:                                                                                                  #  TASK.  EXAMPLES,
010106,000063: 
010107,000064:                                                                                                  #                                OCT     77777           THIS WILL CAUSE AN IMMEDIATE RESTART
010108,000065:                                                                                                  #                                -2CADR  ATASK           OF THE TASK :ATASK:
010109,000066: 
010110,000067:                                                                                                  #                                DEC     200             IF THE TIME OF THE 2 SECONDS SINCE DUMMY
010111,000068:                                                                                                  #                                -2CADR  DUMMY           WAS PUT ON WAITLIST IS UP, IT WILL BEGIN
010112,000069:                                                                                                  #                                                        IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
010113,000070:                                                                                                  #                                                        IT NORMALLY WOULD HAVE BEGUN.
