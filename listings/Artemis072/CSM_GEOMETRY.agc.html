Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

013479,000292:                                                                                                  #  SUBROUTINE NAME:  EARTH ROTATOR       (EARROT1 OR EARROT2)            DATE:  15 FEB 67
013480,000293:                                                                                                  #  MOD NO:  N +1                                                         LOG SECTION:  POWERED FLIGHT SUBROS
013481,000294:                                                                                                  #  MOD BY:  ENTRY GROUP (BAIRNSFATHER)
013482,000295: 
013483,000296:                                                                                                  #  FUNCTIONAL DESCRIPTION:  THIS ROUTINE PROJECTS THE INITIAL EARTH TARGET VECTOR   RTINIT   AHEAD THROUGH
013484,000297:                                                                                                  #        THE ESTIMATED TIME OF FLIGHT. INITIAL CALL RESOLVES THE INITIAL TARGET VECTOR   RTINIT   INTO EASTERLY
013485,000298:                                                                                                  #        AND NORMAL COMPONENTS   RTEAST   AND   RTNORM   . INITIAL AND SUBSEQUENT CALLS ROTATE THIS VECTOR
013486,000299:                                                                                                  #        ABOUT THE (FULL) UNIT POLAR AXIS   UNITW   THROUGH THE ANGLE   WIE DTEAROT   TO OBTAIN THE ROTATED
013487,000300:                                                                                                  #        TARGET VECTOR   RT   . ALL VECTORS EXCEPT   UNITW   ARE HALF UNIT.
013488,000301:                                                                                                  #        THE EQUATIONS ARE
013489,000302:                                                                                                  #                -    -        -                      -
013490,000303:                                                                                                  #                RT = RTINIT + RTNORM (COS(WT) - 1) + RTEAST SIN(WT)
013491,000304: 
013492,000305:                                                                                                  #        WHERE   WT = WIE DTEAROT
013493,000306: 
013494,000307:                                                                                                  #                RTINIT = INITIAL TARGET VECTOR
013495,000308:                                                                                                  #                -        -       -
013496,000309:                                                                                                  #                RTEAST = UNITW * RTINIT
013497,000310:                                                                                                  #                -        -        -
013498,000311:                                                                                                  #                RTNORM = RTEAST * UNITW
013499,000312: 
013500,000313:                                                                                                  #        FOR CONTINUOUS UPDATING, ONLY ONE ENTRY TO EARROT1 IS REQUIRED, WITH SUBSEQUENT ENTRIES AT EARROT2.
013501,000314:
